# 5个AI同时对比：我的工作效率提升300%

> 一个产品经理的真实使用体验

## 🧩 痛点

作为一个产品经理，我每天都要和大量的AI工具打交道：

- 写文案找 **ChatGPT**
- 写代码找 **Claude**
- 搜索答案找 **Perplexity**
- 中文优化找 **Kimi**
- 图片描述找 **豆包**

每天在这些工具之间**反复切换**，同一个问题要问好几遍，标签页开满一屏...

直到我发现了 **AI比一比**。

## 🎯 解决方案

**AI比一比** 让我可以在一个页面里同时打开多个AI：

```
┌─────────────────────────────────────────┐
│  ChatGPT │  Claude  │  Perplexity       │
├──────────┼──────────┼───────────────────┤
│  Kimi    │  豆包    │  Gemini           │
└─────────────────────────────────────────┘
```

**同一问题，一次输入，6个AI同时回答。**

## 💡 真实使用场景

### 场景1：写产品文案
> 同一个需求，让 ChatGPT 写正式版、Claude 写简洁版、Kimi 写年轻化版本——**一次搞定，多选一**。

### 场景2：技术方案选型
> 同一个技术问题，Claude 侧重代码实现、Perplexity 侧重原理分析、ChatGPT 侧重最佳实践——**全方位对比**。

### 场景3：翻译与优化
> 同一段中文，让 Kimi 翻译日文、ChatGPT 翻译英文、豆包 翻译法文——**一键多语种**。

## 🔥 最喜欢的3个功能

1. **快捷键唤醒** - `Ctrl+M` 随时唤出，0.5秒开始干活
2. **划词查询** - 选中网页文字，1秒让多个AI帮我解读
3. **标签页模式** - 新建标签页自动进入对比视图，爽！
