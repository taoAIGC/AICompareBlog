## AI 그림 정말 그렇게 대단할까?

솔직히 말하면, 처음에는 믿지 않았다.

예전의 "그림 잘 못 그리는 사람"이었는데, 간단한 스틱맨도 그리기 힘들었죠. 그런데 AI 이미지 생성 도구를 쓴 지금은 나도 꽤 전문적으로 보이는 이미지를 만들 수 있게 됐습니다.

오늘은 속도 빠른 직접 비교, 실제로 사용해 본 5가지 AI 그림 도구를 보여드릴게요.

## 비교 대상

1. **Midjourney** - 예술성 만점, 디자이너들의最爱
2. **DALL-E 3** - OpenAI 친 아들, GPT 연동 강함
3. **Stable Diffusion** - 로컬 설치版 돈 아끼기王
4. **Adobe Firefly** - PS 생태福利
5. **Leonardo.ai** - 기능 전면적인 후발 주자

## 비교 항목

- 생성 품질
- 조작 편리함
- 무료 크레딧
- 커스터마이징 가능성
- 생성 속도
- 가격

---

## 1. Midjourney - 예술감의 천장

**장점**:
- 예술감满满, 나오면 그저高级感
- 프롬프트 이해能力强
- 커뮤니티 활성화, 남이 어떻게 프롬프트 쓰는지 参考 가능
- 지속 업데이트, 버전 빠르게 진화
- V6 버전부터 텍스트 생성 지원

**단점**:
- 과학上网 필요 (VPN)
- Discord 조작이 초보에게不太 친화적
- 무료 크레딧 소진 후 유료
- 저작권 문제가 좀 애매

**적합 시나리오**: 디자이너, 예술 창작, 소셜 미디어 이미지

**가격**: 무료 체험, 유료版 10-30달러/월

---

## 2. DALL-E 3 - GPT 연동 最強

**장점**:
- GPT-4와 깊이 통합, 자연어로 대화식으로 생성 가능
- 이해能力强, 좀처럼 빗나가지 않음
- 저작권 상대적으로 명확, 상업적 사용 위험 적음
- 생성 속도 빠름
- 안전 필터 있어서 원치 않는 콘텐츠 생성 방지

**단점**:
- 과학上网 필요 (VPN)
- 예술감이 Midjourney보다 못함
- 무료 크레딧 적음
- 커스터마이징 정도 상대적으로 낮음

**적합 시나리오**: 제품 이미지, 콘텐츠 창작, 정밀한 제어 필요한场景

**가격**: 무료版 제한, 유료版 20달러/월 (GPT Plus 포함)

---

## 3. Stable Diffusion - 로컬 설치版 돈 아끼기王

**장점**:
- 완전 무료 (로컬 설치)
- 내 컴퓨터에서 돌릴 수 있어서 프라이버시 보장
- 커스터마이징 정도 매우 높음, 플러그인 풍부
- 인터넷 연결 없이도 사용 가능
- 커뮤니티 활성화, 모델 다양함

**단점**:
- 하드웨어 요구사항 높음, 좋은 그래픽카드 필요
- 설치와 설정이 좀 귀찮음
- 출력 품질이 모델과 프롬프트에 따라 달라짐
- 초보 입문 장벽 높음

**적합 시나리오**: 중급 이상 사용자, 프라이버시 중시, 대량 생성 필요

**가격**: 무료 (하드웨어 비용 별도)

---

## 4. Adobe Firefly - PSdala의 봄

**장점**:
- Adobe全家桶에 통합
- 저작권 문제 가장 명확하게 처리
- 상업용 보안성 높음
- Photoshop과 긴밀하게 연동
- 자연어로 이미지 편집 가능

**단점**:
- 생성 품질이 Midjourney 못함
- 기능 상대적으로 제한적
- Adobe 구독 필요
- 국내 접속 불안정

**적합 시나리오**: 디자이너, Adobe 사용자, 비즈니스 프로젝트

**가격**: Adobe 구독 필요, 단독版 4.99달러/월부터

---

## 5. Leonardo.ai - 후발 주자

**장점**:
- 기능 전면적, 있어야 할 것 다 있음
- 무료 크레딧 상대적으로많음
- 조작 인터페이스 친화적
- 다양한 모델 지원
- 모바일 앱 있음

**단점**:
- 예술감이 Midjourney보다稍逊
- 커뮤니티가 Midjourney만큼 활발하지 않음
- 저작권 약관이 종종 바뀜
- 국내 사용 불안정

**적합 시나리오**: 초보, 무료 크레딧 원하는 사용자

**가격**: 무료版 제한, 유료版 10달러/월부터

**초보 입문**: Leonardo.ai로 시작해보기

## 팁: AI 그림 더 효율적으로 쓰기

1. **프롬프트는 구체적으로**: "고양이 한 마리 그려줘"만 하지 말고, "창가에 앉은 오렌지색 고양이, 옆에서 햇빛 비추고, 따뜻하고 힐링되는 스타일, 8K 초고화질" 이렇게 구체적으로 설명

2. **참고 이미지 활용**: 대부분의 도구에서 참고 이미지 업로드 지원하는데, 이거 하면 AI가 원하는 느낌 더 쉽게 이해함

3. **여러 장 생성해서 고르기**: 한 번에 4장 생성해서 가장 마음에 드는 거 고르고, 그거 계속 미세 조정

4. **타인 프롬프트 배우기**: Midjourney 커뮤니티에 사람들이 프롬프트 공유하는 것 많으니 참고해서 배울 수 있음

5. **후처리 필수**: AI 생성 뒤에 PS나其它 앱으로 수정하면 훨씬 좋아짐
