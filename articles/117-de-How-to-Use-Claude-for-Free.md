## Wie heißt die kostenlose Version von Claude?

Claude hat drei Versionen:

- **Claude Haiku** (kostenlos)
- **Claude Sonnet** (kostenlos)
- **Claude Opus** (nur für Pro-Version)

Für den täglichen Gebrauch reicht Haiku vollkommen aus.

Die Eigenschaften von Haiku:
- Schnelle Reaktionszeit
- Geeignet für einfache Gespräche
- Perfekt für Texte, Code-Korrektur, Chat
- Komplett kostenlos, unbegrenzt

## Was ist der Unterschied zur Bezahlversion?

Ehrlich gesagt, nicht viel.

**Alltägliche Nutzung**:
- Die kostenlose Version reicht
- Schnelle Reaktionszeit
- Ähnliche Erfahrung

**Komplexe Aufgaben**:
- Kostenlose Version vereinfacht Antworten möglicherweise
- Bezahlversion detaillierter und tiefer
- Lange Texte处理bei Bezahlversion besser

**Direkter Vergleich**:

| Szenario | Kostenlose Version | Bezahlversion |
|----------|-------------------|---------------|
| Texte schreiben | ✅ Reicht aus | ✅ Besser |
| Code schreiben | ✅ Reicht aus | ✅ Stärker |
| Langer Text | ⚠️ Begrenzt | ✅ Unbegrenzt |
| Dateianalyse | ❌ Nicht unterstützt | ✅ Unterstützt |
| Reaktionsgeschwindigkeit | ✅ Schnell | ✅ Noch schneller |

Fazit: Für den Alltag reicht die kostenlose Version, Power-User nehmen die Bezahlversion.

## Einschränkungen der kostenlosen Version

### Nachrichtenanzahl

Die kostenlose Version hat ein Nachrichtenlimit:
- Claude Haiku: Alle paar Stunden wird zurückgesetzt
- Claude Sonnet: Etwas mehr als Haiku
- Genaue Anzahl unbekannt, reicht aber für den Alltag

### Peak-Beschränkungen

In Stoßzeiten kann es Wartezeiten geben:
- An Werktagen tagsüber meist normal
- Abends in Stoßzeiten kurz warten
- Meist nur wenige bis dutzende Sekunden

### Funktions-Einschränkungen

Die kostenlose Version kann nicht:
- Dateien hochladen und analysieren
- Bildfunktionen nutzen
- Auf bestimmte Premium-Funktionen zugreifen

## Tipps zur Nutzung der kostenlosen Version

### Tipp 1: Gespräche trennen

Für verschiedene Themen neue Gespräche starten:
- Ein Gespräch für Code
- Ein Gespräch für Texte
- Ein Gespräch zum Chatten

Bleibt übersichtlicher.

### Tipp 2: Wichtige Gespeicherte Gespräche sichern

Nützliche Gespräche lokal "kopieren und speichern".

### Tipp 3: Nachfragen lernen

Wenn beim ersten Mal nicht klar wurde, weiterfragen:
- "Kannst du das genauer erklären?"
- "Gib mir ein paar Beispiele"
- "Sag das andersrum"

### Tipp 4: Länge kontrollieren

Bei langen Artikeln, in Teilen anfordern:
- "Schreib zuerst den ersten Teil"
- "Dann den zweiten Teil"
- "Zum Schluss die Zusammenfassung"

So gibt's keine Timeouts.
