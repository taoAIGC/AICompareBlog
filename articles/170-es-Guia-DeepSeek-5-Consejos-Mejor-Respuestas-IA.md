## Consejo 2: Cómo escribir la palabra clave

### No lo uses por mucho tiempo

DeepSeek no es lo mismo que ChatGPT.

ChatGPT escribe una larga lista de palabras clave y las lee todas.

Si escribe DeepSeek demasiado tiempo, fácilmente se desviará.

**Escritura correcta**:
```
Escribe un artículo sobre comunicación en el lugar de trabajo, de unas 1000 palabras. El público objetivo son los recién llegados que acaban de incorporarse al trabajo. Debe ser práctico y no demasiado teórico.
```

**Forma incorrecta de escribir**:
```
Usted es un experto senior en recursos humanos con 10 años de experiencia en contratación y es bueno manejando problemas de relaciones en el lugar de trabajo. Por favor escriba un artículo de acuerdo con los siguientes requisitos: primero... segundo... luego... finalmente...
```

Es demasiado largo, DeepSeek leyó el final y olvidó el principio.

### Pregunta paso a paso

Haga preguntas complejas por separado.

Por ejemplo, si quieres escribir un artículo:

Paso uno:
```
Ayúdenme a hacer un resumen del artículo "Habilidades de comunicación en el lugar de trabajo", 5 partes
```

Después de que DeepSeek diera el esquema:

Paso dos:
```
Basado en el primer esquema, escribe un párrafo detallado de 200 palabras.
```

Paso tres:
```
Continúe escribiendo la segunda parte del esquema.
```

Esto funciona mucho mejor que hacer todas las preguntas a la vez.

### Borrar formato

DeepSeek a veces puede exagerar y escribir un montón de cosas que no necesitas.

Dígale exactamente qué formato desea.

```
Muéstrelo en forma de tabla. Cada elemento de comparación debe incluir: función, rendimiento de DeepSeek, rendimiento de ChatGPT
```

```
Por favor explique en puntos, no más de 5 ítems, cada ítem no debe exceder las 50 palabras.
```

No será gratis jugar.

## Consejo 3: uso basado en escenarios

### Escribir copia

DeepSeek es muy bueno escribiendo textos publicitarios en chino.

Especialmente el tipo con los pies en la tierra.

Por ejemplo, escribir un texto para Momentos:
```
Ayúdame a escribir un círculo de amigos, diciendo que mis colegas me dieron té con leche y pastel para el té de la tarde esta tarde. Sea un poco gracioso.
```

DeepSeek te dará una versión interesante.

### Escribir código

También se puede utilizar al escribir código.

Pero el efecto real medido no es tan bueno como el de Claude.

La escritura del código de Claude está más estandarizada y sus comentarios son más claros.

DeepSeek a veces tiene problemas menores al escribir código, por lo que se recomienda comprobarlo dos veces.

###Escribe un artículo

DeepSeek puede escribir artículos de cuentas oficiales, respuestas de Zhihu y redacción publicitaria de Xiaohongshu.

Especialmente para contenido que requiere "sentido de Internet", el sentido del lenguaje de DeepSeek es muy bueno.

### traducir

La traducción de DeepSeek es más natural que la de Google Translate.

No es una traducción automática rígida, pero se lee como habla humana.

```
Ayúdenme a traducir este párrafo del inglés al chino. Debe ser natural y no literal.
```

Funciona muy bien.

### Lluvia de ideas

¿No se te ocurre un tema para elegir? ¿No sabes cómo tomar una decisión?

Deje que DeepSeek le ayude a generar ideas.

```
Soy un blogger en el lugar de trabajo. Escribo principalmente sobre crecimiento en el lugar de trabajo y habilidades de comunicación. Por favor dame 10 temas para elegir.
```

Te dará un montón de ideas.

## Consejo 4: Guía para evitar trampas

### No confíes demasiado

DeepSeek a veces dice tonterías graves.

Fabrica datos y citas para parecer profesionales, pero en realidad son falsos.

**Solución**:
- Verificación manual de información importante.
- Implica verificación multiángulo del conocimiento profesional.
- No copiar directamente al documento oficial

### Temas delicados

DeepSeek tiene revisión de contenido y hay algunas preguntas que no responde.

Por ejemplo:
- Cómo evitar el pago
- Cómo descifrar software
- Acontecimientos históricos sensibles

Rechazará estas preguntas o le dará una charla oficial.

### Restricciones contextuales

La ventana de contexto de DeepSeek no es tan larga como la de Claude.

Al final de escribir un artículo extenso, es posible que olvide lo dicho anteriormente.

**Solución**:
- Escribe en secciones, cada sección debe tener una extensión máxima de 2000 palabras.
- Repetir información importante al principio de cada párrafo.
- Resuma manualmente el contenido anterior y aliméntelo si es necesario.

### No preguntes sobre cosas que son demasiado nuevas.

Los datos de entrenamiento de DeepSeek tienen fecha de vencimiento.

Quizás no sepa qué sucederá después de 2024.

Pregúntele "¿Qué gran cosa pasó recientemente?" y le dará una respuesta obsoleta.

## Consejo 5: uso avanzado

### Palabra de aviso del sistema

DeepSeek admite palabras de aviso del sistema.

Haga clic en Configuración en la esquina superior derecha para configurar el comportamiento predeterminado de la IA.

Por ejemplo:
```
Eres un consultor de negocios profesional. Responda preguntas de manera concisa y efectiva, y utilice datos y casos para respaldar sus opiniones.
```

Una vez configurada, todas las conversaciones posteriores tendrán efecto.

### Plantilla de conversación

Las palabras útiles se pueden guardar como plantillas.

DeepSeek actualmente no tiene una función de plantilla oficial, pero puedes:

1. Escríbelo en un documento
2. Copie y pegue cuando sea necesario
3. O toma una captura de pantalla y guárdala.

### llamadas API

Los desarrolladores pueden acceder a través de API.

La API de DeepSeek es más barata que OpenAI.

La documentación está claramente escrita y es amigable para los desarrolladores.

### Implementación local

El código abierto significa que puede descargar el modelo y ejecutarlo localmente.

Requisitos de configuración:
- Tarjeta gráfica con al menos 16 GB de memoria de vídeo
- Memoria 32GB o más
- Espacio en disco duro 100 GB+

Las computadoras comunes no pueden ejecutarlo, por lo que se recomienda utilizar servicios en la nube.
