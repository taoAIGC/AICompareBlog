## 数据分析的AI组合

### 场景一：SQL编写

**输入提示词**：
> "写一个SQL查询，统计最近7天每天的活跃用户数，按日期降序排列，用户表名是users，活跃定义是当天有登录行为。"

**ChatGPT版**：
```sql
SELECT 
    DATE(created_at) as date,
    COUNT(DISTINCT user_id) as dau
FROM user_actions
WHERE action = 'login'
    AND created_at >= DATE_SUB(CURDATE(), INTERVAL 7 DAY)
GROUP BY DATE(created_at)
ORDER BY date DESC;
```

**Claude版**：
```sql
WITH daily_active AS (
    SELECT 
        DATE(created_at) as date,
        COUNT(DISTINCT CASE WHEN action = 'login' THEN user_id END) as dau
    FROM user_actions
    WHERE created_at >= DATE_SUB(CURDATE(), INTERVAL 7 DAY)
    GROUP BY DATE(created_at)
)
SELECT 
    date,
    dau,
    LAG(dau) OVER (ORDER BY date) as prev_dau,
    ROUND((dau - LAG(dau) OVER (ORDER BY date)) / NULLIF(LAG(dau) OVER (ORDER BY date), 0) * 100, 2) as pct_change
FROM daily_active
ORDER BY date DESC;
```

**Perplexity版**：
```

       COUNT(DISTINCT user_id) as dau
FROM events
WHERE event_type = 'login'
  AND event_time >= NOW() - INTERVAL '7' DAY
GROUP BY 1
ORDER BY 1 DESC
```

### 场景二：Python数据分析

**输入提示词**：
> "用Python写一段代码，读取CSV文件，计算每个用户的购买频次、客单价，并找出复购率最高的10个用户。"

**ChatGPT版**：
```python
import pandas as pd

# 读取数据
df = pd.read_csv('orders.csv')

# 计算购买频次和客单价
user_stats = df.groupby('user_id').agg(
    purchase_frequency=('order_id', 'count'),
    total_spending=('amount', 'sum'),
    purchase_count=('order_id', 'nunique')
).reset_index()

user_stats['avg_order_value'] = user_stats['total_spending'] / user_stats['purchase_count']

# 计算复购率（有多次购买的用户占比）
repeat_rate = (user_stats['purchase_count'] > 1).sum() / len(user_stats) * 100

# 复购率最高的10个用户
top_10 = user_stats.nlargest(10, 'purchase_count')

print(f"复购率: {repeat_rate:.2f}%")
print("\n复购率最高的10个用户:")
print(top_10)
```

### 场景三：数据可视化建议

**输入提示词**：
> "我想做一个用户留存分析的可视化图表，展示次日留存、7日留存、30日留存，应该用什么图表，怎么设计？"

**Claude版**：
```
推荐使用【折线图】+【热力图】组合：

1. 折线图（趋势分析）
   - X轴：天数（Day 1, Day 7, Day 30）
   - Y轴：留存率（%）
   - 多条线：不同获客渠道/用户群体
   - 用途：观察留存趋势变化

2. 热力图（细节分析）
   - X轴：获客日期
   - Y轴：天数
   - 颜色深浅：留存率高低
   - 用途：定位具体日期的留存异常

3. 组合建议
   - 日常监控：看折线图
   - 问题排查：看热力图
```
