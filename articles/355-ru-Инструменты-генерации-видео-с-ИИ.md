## 1. Runway Gen-3 - стандарт индустрии

**Плюсы**:
- Стабильное качество видео, хорошая репутация
- Полный функционал: генерация, редактирование, расширение
- Сильный контроль, точное управление камерой
- Активное сообщество, много уроков
- Постоянные обновления

**Минусы**:
- Нужен VPN
- Ограниченный бесплатный лимит
- Высокие требования к промптам, новичкам сложно
- Недешево

**Подходит для**: Профессиональные создатели, блогеры, реклама

**Цена**: Бесплатно ограничено, 12-35$/мес

---

## 2. Pika - удобно для новичков

**Плюсы**:
- Простое управление, чистый интерфейс
- Быстрая генерация, результат за минуты
- Удобно для новичков, не нужно учить сложные навыки
- Поддерживает разные стили
- Есть бесплатный лимит для пробы

**Минусы**:
- Качество видео хуже чем у Runway
- Ограниченная настройка
- Длинные видео часто ломаются
- Нестабильный доступ в Китае

**Подходит для**: Новички, контент для соцсетей, быстрые прототипы

**Цена**: Бесплатно ограничено, 10-28$/мес

---

## 3. Luma Dream Machine - для ценителей качества

**Плюсы**:
- Высокая чёткость видео
- Хорошая физическая симуляция, естественное движение объектов
- Сильное понимание промптов
- Поддерживает текст и изображение в видео
- Плавное движение

**Минусы**:
- Медленнее генерирует
- Ограниченный бесплатный лимит
- Нужен VPN
- Иногда генерирует странный контент

**Подходит для**: Фокус на качество, создание контента, естественное движение

**Цена**: Бесплатно ограничено, ~20$/мес

## 4. Sora - большие ожидания

**Плюсы**:

- Генерирует длинные видео (до 60 секунд)

- Цена неизвестна
- Много ограничений

**Цена**: Не объявлена

## 5. Kling AI - тёмная лошадка

**Плюсы**:

- Хорошая поддержка китайского
- Относительно щедрый бесплатный лимит

- Быстрые обновления

**Минусы**:
- Новее, меньше накопленного опыта
- Меньше ресурсов сообщества чем у международных
- Некоторые функции ещё улучшаются
- Международная версия может отличаться

**Подходит для**: Китайские пользователи, хотят бесплатные кредиты, быстрый результат

**Цена**: Бесплатно ограничено, выгодные платные планы

## Продвинутые советы: используем видео-ИИ эффективнее

1. **Будьте конкретны в промптах**: Не говорите просто "кошка бежит", попробуйте "рыжий кот бежит слева направо, фон — весенний луг, лепестки летят по ветру, кинематографическое качество"

2. **Сначала изображение, потом видео**: Многие инструменты поддерживают создание ключевого изображения, потом конвертацию в видео — так легче контролировать результат

3. **Генерируйте по сегментам**: Длинные видео легко ломаются, генерируйте по частям и объединяйте

4. **Комбинируйте с традиционным монтажом**: Сгенерированные видео часто нужно дорабатывать в CapCut, добавлять субтитры, корректировать цвет

5. **Пробуйте больше, жалуйтесь меньше**: ИИ быстро развивается, то что сегодня не работает завтра может улучшиться — экспериментируйте

## Заметки

- Авторские права на видео с ИИ ещё обсуждаются, будьте осторожны с коммерцией
- При генерации видео с людьми избегайте нарушений
- Большинству инструментов нужен VPN
- Платите по необходимости после окончания бесплатного лимита
- Генерация видео как "гача" — генерируйте много раз и выбирайте лучшее
- Инструменты быстро обновляются, информация может устареть
