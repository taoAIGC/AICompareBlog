## 従来法 vs AI法

### 従来法（3時間以上）

1. Baiduで「XX業界の現状」を検索 → 画面一面広告
2. 記事を一つずつ開く → 頭痛い
3. ドキュメントにコピペ → フォーマットぐちゃぐcha
4. 要約まとめる → 2時間
5. レポート書く → 1時間

**結果**: 深夜まで残業、品質悪い可能性高い。

### AI法（1時間）

1. 4つのAIに同時に別角度の質問 → 15分
2. AIが自動で要点を整理 → 10分
3. 人間が選択・結合 → 15分
4. フォーマット調整 → 20分

**結果**: 定時退勤、品質悪くない。

## 実践ステップ

### ステップ1：調査ニーズを明確化

上司が「業界調查」と言ったら、何が必要か具体的に聞く。

**魔法のフレーズ**:
> "上司님、具体的にどこ为重点にしますか？市場規模、競争環境、発展趋势、または特定企業？"

2つの利点:
1. プロフェッショナルで真面目そう
2. 方向明確、無駄な仕事が減る

### ステップ2：複数のAIに同時に質問

AIツールを開いて、4つの角度の質問を入力:

**質問1（市場規模）**:
> 中国のXX業界の市場規模、成長率、過去3の発展トレンドを紹介してください。

**質問2（競争環境）**:
> 中国のXX業界の主要プレイヤーは？市場シェアのDistributionは？

**質問3（発展トレンド）**:
> 今後3-5年、XX業界の新技術、新トレンド、新機会は？

**質問4（課題・リスク）**:
> XX業界は現在どんな課題やリスクに直面？政策の影響は？

**なぜ4つのAIに聞く？**
各AIの得意分野不同。ChatGPT包括的、Claude論理的、DeepSeek中国語理解 good、Gemini思わぬ惊喜をくれるかも。

### ステップ3：AIが整理出力

4つの回答を得たら、AIに速く整理させる:

**プロンプト**:
> 以下の内容を含む業界調査レポートが必要です：
> 1. 業界概要（市場規模、成長率）
> 2. 競争環境（主要プレイヤー、市場シェア）
> 3. 発展トレンド（今後3-5年）
> 4. 結論と提案
> はっきりしたフォーマットで、800字程度で整理してください。

### ステップ4：人手での最適化

AIは強いが、最後は人手が必要:

**チェックリスト**:
- [ ] データソース妥当？
- [ ] ロジック流畅？
- [ ] フォーマット清晰？
- [ ] タイポ check？
- [ ] 重要データ太字？

##  реальный пример

前に紧急タスクを受けた—"スマートホーム業界"を調査。

従来法なら熬夜かもしれないが、AI法使えば：

- 5分：4つのAIに別角度の質問
- 10分：4つの异なる视角の回答得到
- 15分：AIに800字レポートにまとめさせる
- 10分：人手检查・フォーマット調整
- **合計40分**、完成提交！

上司看完说："不错、挺快的！"

## 上級テクニック

### テクニック1：補足質問

AIの回答が不十分なら：
> "先のXXデータについて、更詳細に？具体例は？"

### テクニック2：相互検証

不同AI回答冲突的话：
> "ChatGPTは市場規模50B、DeepSeekは80B言う—実際のソース確認必要。"

### テクニック3：構造化出力

AIに欲しいフォーマットで出させる：
> "トップ5競合と市場シェアをテーブル形式でリストしてください。"

## まとめ

緊急の調査依頼があったら、4ステップ覚えて：

1. **ニーズ明確化** — 上司が本当に欲しいものを先に聞く
2. **複数AIに聞く** — 異る角度から全面的情报获取
3. **AI整理** — 速やかに下書き生成
4. **人手最適化** — 品質确保のため检查・調整

关键是不慌—AI是你的底气。
