## なぜ履歴書が大切なの？

履歴書はあなたの第一印象カード。HRは履歴書を审查するのに平均6-7秒しか使わない。その6秒間で注目を得られなければ、その後能力がっても無駄。

多くの人は能力があるのに、履歴書が平凡すぎて、HRがあなたの亮点看不到。AI就能帮你解决这个问题。

## AIで履歴書を書く具体的な方法

### 方法1：AIに 경험을 정리してもらう

很多人写简历最大的问题不是经历不够，而是不知道该怎么表达。

**操作步骤**：
1. 把你的工作经历、项目经历整理成文字
2. 告诉AI你的岗位方向
3. 让AI帮你重新梳理和表达

**プロンプト例**：
```
产品经理职位に応募这是我工作经验：
1. 会社のAPPの需求调研和機能规划を担当
2. 5人チームを率いて3つの重要プロジェクトを完了
3. ユーザー定着率が20%向上

これらの経験をより専門的な言葉で再描述し、私の能力を際立たせてください。
```

### 方法2：AIに表現を最適化してもらう

同样的内容，不同的表达方式，效果完全不同。

**Before vs After**：

❌ 修正前：
```
製品の相关业务を担当、色んな機能做了、ユーザー数も向上した。
```

✅ 修正後：
```
为核心产品的功能规划与迭代主导。通过用户调研数据分析，推动3项核心功能上线，帮助DAU提升25%。

```

**プロンプト例**：
```
この履歴書描述をより専門的かつデータ导向に最適化してください：
[履歴書のの内容を貼り付け]
要件：1. 具体的なデータを追加 2. 専門用語を使用 3. 個人の貢献を際立たせる
```

### 方法3：AIに岗位要求をマッチングさせる

投简历最怕的是岗位要求和你的经历不匹配。

**プロンプト例**：
```
応募职位的要求是：
1. 3年以上产品经验
2. 项目管理能力
3. 数据分析能力
4. 用户增长经验

私の经验是：
[経験を貼り付け]

私の匹配度を分析し、履歴書で強化すべき部分を指摘してください。

## 常用的履歴書最適化コマンド

### 1. 成果を数量化
```
「ユーザー満足度を向上させた」をより具体的なデータ描述に変更
```

### 2. 動詞をグレードアップ
```
「担当」、「やった」を「主導」、「推進」、「達成」などの更有力な動詞に変更
```

### 3. 構造最適化
```
この経験を「背景-アクション-結果」構造で再整理
```

### 4. キーワードマッチング
```
この履歴書を以下の募集要件にマッチするよう最適化：[募集要件を貼り付け]
```

## 注意事项

### ❌ 这样做不行

1. **AIに全部書はせない** - AIが書いた内容は 너무千篇一律，缺乏个人特色
2. **テンプレートをコピーしない** - テンプレート的履歴書はHRにすぐばれる
3. **事実を夸张しない** - 面试时被追问就会露馅
4. **フォーマットを無視しない** - 良いフォーマットはHRが見て舒服

### ✅ 应该这样做

1. **まずは自分で書く** - 主要经历和亮点整理好
2. **AIに最適化させる** - 表現を最適化、亮点を際立たせ、数量化
3. **自分で確認する** - 内容真实、論理が通るか確認
4. **応募先によって調整** - 一つの履歴書を всем投递

## 常见问题

###が書いた履歴書は Q1：AI会不会太千篇一律？
A：あるので、AIが書いたものに自分の个人特色や真实经历を必ず追加。AIは辅助而不是替代。

### Q2：履歴書の长さ有什么要求？
A：应届生1页，有经验的人2页为宜。太长的话HR没时间看。

### Q3：履歴書で最も重要なのは什么？
A：最重要的是让HR在6秒内看到你的亮点。データ的表达和突出核心成就是最重要的。

### Q4：AI能把帮你做履歴書模板吗？
A：可以，但建议还是用一些专业的履歴書模板网站，AI主要帮你优化内容。

## 总结

AI写履歴書の正确姿势：
1. **まずは自分で書く** - 基本内容を整理
2. **AIに最適化させる** - 表現を最適化、数量化
3. **自分で把关** - 真实で个人特色があることを確認
4. **応募先によって调整** - 応募先ごとに異なるバージョン
