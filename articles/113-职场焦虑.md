## 4个AI工具写邮件实测对比

### 测试场景

给甲方写一封催款邮件：语气要客气，但不能太软；要专业，但不能太生硬。

### 1. ChatGPT - 成熟稳重型

```
请帮我写一封催款邮件给甲方客户。背景是我们已经交付
项目两个月了，合同约定的是验收后30天付款。
语气要友好但专业，催促对方尽快安排付款。
```

**优点**：英文邮件特别强，语气把控到位
**缺点**：中文语境下偶尔有点翻译腔
**适用**：外贸、商务英文邮件

### 2. Claude - 逻辑清晰型

Claude写邮件的特点是结构清晰，每段话的目的很明确。

```
请写一封中文商务邮件，主题是催促项目款项。
邮件应该包含：1）感谢对方的合作；2）说明已交付内容；
3）提到合同约定的付款时间；4）礼貌催促。
```

**优点**：逻辑性强，结构清晰
**缺点**：风格偏保守，创新性不足
**适用**：正式报告、申请类邮件

### 3. DeepSeek - 接地气型

DeepSeek在中文语境下表现很接地气，写出来的邮件更像中国人说的话。

```
帮我写一封催款邮件，我们是乙方，已经做完项目两个月了，
客户那边还没付款。语气要好一点，不能得罪人，
但是要让他知道我们着急收款。
```

**优点**：中文表达自然，不生硬
**缺点**：需要给更具体的指令
**适用**：中文日常商务沟通

### 4. Kimi/豆包 - 快速模板型

这两个工具更适合快速生成邮件模板，然后人工微调。

**优点**：响应快，模板多
**缺点**：深度定制能力弱
**适用**：简单通知、常规回复

## 不同场景的AI邮件指令

### 场景1：催客户/甲方

```
请帮我写一封【催促付款】邮件。
背景：项目已交付[具体内容]，距合同约定付款日期已超过[X]天。
要求：
- 语气友好但有紧迫感
- 提及双方的良好合作关系
- 明确表达希望对方尽快处理
- 留下联系方式方便沟通
```

### 场景2：跨部门协作

```
请帮我写一封【跨部门协作】邮件。
背景：我是[部门]，需要[其他部门]协助完成[具体任务]。
要求：
- 说明任务的重要性和紧迫性
- 明确需要对方提供什么支持
- 给出明确的时间节点
- 语言要客气，不要有命令感
```

### 场景3：向上级汇报

```
请帮我写一封【工作汇报】邮件。
背景：[项目/任务]已完成[进度]，遇到[问题/困难]。
要求：
- 先说结论，再展开细节
- 突出关键成果和数据
- 诚恳说明问题和解决方案
- 结尾给出下一步计划
```

### 场景4：拒绝/推辞类邮件

```
请帮我写一封【委婉拒绝】邮件。
场景：[具体情况]。
要求：
- 先表达感谢或理解
- 再说明无法配合的原因
- 提供替代方案（如果有）
- 保持友好但态度明确
```

## AI写邮件的避坑指南

### ❌ 别直接用AI生成的原文

AI写的邮件是"标准答案"，但你的情况是"具体情况"。一定要：
- 替换具体的项目名称、时间、数字
- 加入你个人的风格和语气
- 检查是否有不准确的信息

### ❌ 别让AI处理敏感内容

涉及机密、纠纷、投诉、辞职这类敏感邮件，AI帮不了你。
这种邮件需要个人承担后果，还是自己写比较稳妥。

### ❌ 别忘了检查格式

AI生成的邮件可能在格式上有问题：
- 收件人、抄送人是否正确
- 附件是否真的附上了
- 签名档是否更新了

## 进阶技巧：训练你的AI邮件助手

把常用的邮件场景整理成文档，让AI学习你的风格：

```
我的邮件写作偏好：
1. 开头喜欢用"您好"而不是"你好"
2. 段落控制在3-4行
3. 常用表达：[收集一些你喜欢的句式]
4. 结尾通常是"期待您的回复"
```

把这个喂给AI，以后它写出来的邮件就更像你写的。
