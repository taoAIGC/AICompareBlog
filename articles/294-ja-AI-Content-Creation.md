## AI作成できること

### 1. 下書き作成
どこから手をつけていいか分からない時は、まずAIに下書きを書いてもらって、それから自分で修正责任编辑，比从头开始写快得多。

### 2. 書き直しと最適化
書いたものはあんまりよくないと思われる場合は、AIに最適化させてくれ，比自己改来改去效率更高。

### 3. ブレインストーミング
何的话题にしたらいいか分からない場合は、AIにアイデアを出してもらえる、10個8個主题、总有一个能启发你。

### 4. タイトル作成
良いタイトルは記事の命運を左右する。AIに複数のタイトル案を出してもらえる。

### 5. 结尾提升
结尾总是写不好，AI可以帮你总结升华，让文章更有深度。

## AI作成常见内容类型的写法

### 1. 公众号文章

**プロンプト例**：
```
你是资深新媒体编辑，帮我写一篇公众号文章。

主题：[主题]
目标读者：[读者群体]
文章结构：
1. 开头：用[故事/数据/问题]引入
2. 主体：3个分论点，每个配案例
3. 结尾：总结+行动召唤

要求：
- 语言风格：[风格]
- 字数1500字左右
- 要有具体案例和数据
```

### 2. 小红书文案

**プロンプト例**：
```
帮我写[数量]条小红书种草文案。

产品/话题：[产品或话题]
目标人群：[人群]
风格：[风格]
要求：
- 开头要有吸引力
- 加入真实使用感受
- 结尾要有互动引导
- 每条[字数]字以内
```

### 3. 抖音脚本

**プロンプト例**：
```
帮我写一个抖音视频脚本。

视频主题：[主题]
时长：[秒]
目标人群：[人群]
风格：[风格]
结构：
1. 黄金3秒：如何抓住注意力
2. 内容主体：[内容]
3. 结尾引导：如何引导点赞评论
```

### 4. 朋友圈文案

**プロンプト例**：
```
帮我写一条朋友圈文案。

场景：[工作/生活/产品发布]
目的：[如种草/分享/引流]
要求：
- 简洁有吸引力
- 有互动性
- [字数]字以内
```

## AI作成使用技巧

### 技巧1：给AI设定角色

**错误**："帮我写篇文章"

**正确**："你是一个有10年经验的资深新媒体编辑，擅长写干货类文章..."

### 技巧2：给足背景信息

**错误**："写个文案"

**正确**："我们是一个做职场教育的公众号，目标用户是25-35岁的职场人..."

### 技巧3：分步骤处理

**错误**："帮我写一篇完整的公众号文章"

**正确**：
1. 先让AI出3个选题
2. 选好选题后，让AI写大纲
3. 大纲确认后，让AI写初稿
4. 初稿完成后，让AI优化

### 技巧4：指定输出格式

**错误**："总结一下"

**正确**："用表格形式总结以下内容的优缺点"

## 注意事项

### ❌ 不要这样做

1. **不要直接照抄** - AI写的务必要自己修改
2. **不要忽视事实** - AI会胡编数据，一定要核实
3. **不要忽略平台特性** - 每个平台风格不同
4. **不要过度依赖** - AI是辅助，不是替代你的思考

### ✅ 应该这样做

1. **先自己思考** - 核心观点要自己想，AI帮你表达
2. **仔细核实** - 数据、案例要核实真假
3. **加入个人特色** - 加入自己的经历、风格
4. **持续学习** - 多看优秀内容

## 总结

AI创作的核心心法：

1. **AI是辅助不是替代** - 核心想法要自己想
2. **给AI足够的背景** - 背景越详细，输出越准确
3. **一定要修改** - AI写的必须自己修改调整
4. **持续优化** - 不断调教AI，让它更懂你的风格

用好AI，内容创作效率翻倍不是梦！
