## AIメール作成が業務必需になった理由

毎日メール作成にかかる時間は？少なくとも30分、時には2〜3時間。

クライアントへの督促、報告業務、資源申請、部門間連携...每メールには細心の注意が必要で、間違いが怖い。

AIにメール作成を依頼：1時間かかるものが10分で完了。これはサボりではなく、効率化。

## 4つのAIツールでメール作成を比較検証

### テストシナリオ

クライアントへの入金督促メール：丁寧なトーンだが弱すぎず、プロフェッショナルだが硬すぎない。

### 1. ChatGPT - 安定型

```
クライアントへの入金督促メールを作成してください。
背景：プロジェクトを2ヶ月前に納品し、契約では受領後30日以内の入金となっております。
トーンはfriendlyですがprofessional、相手に入金尽快対応を促す内容にしてください。
```

**長所**：英文メールEspecially強い、トーンコントロール出色
**短所**：中国語の文脈でたまに変な翻訳調
**おすすめ**：貿易、ビジネス英文メール

### 2. Claude - 論理型

Claudeのメール特点是論理的で明確、各段落の目的が明確。

```
中式ビジネスメール、件名はプロジェクト入金の督促。
内容：1)協力への感謝、2)納品内容の説明、3)契約上の入金期日の言及、4)礼貌な督促
```

**長所**：論理性強い、構造清晰
**短所**：スタイルが保守的で innovation 少なめ
**おすすめ**：正式なレポート、申請メール

### 3. DeepSeek - 接地気型

DeepSeekは中国語文脈でとても自然、中国人が書くメールに近い。

```
入金督促メール帮我写，我们是乙方，已经做完项目两个月了，客户还没付款。
トーンは好一点，不能得罪人，但是要让他知道我们着急收款。
```

**長所**：中国語表現自然、硬くない
**短所**：より具体的な指示が必要
**おすすめ**：中式日常ビジネスコミュニケーション

### 4. 快速テンプレート型 (Kimi/Doubao)

这两个工具更适合快速生成邮件模板，然后人工微调。

**長所**：対応速い、テンプレート 많다
**短所**：深カスタマイズ弱い
**おすすめ**：简单通知、日常返信

## シーン別AIメール指示

### シーン1：クライアント/パートナーへの督促

```
【入金督促】メール帮我写。
背景：プロジェクト已納品[具体内容]、契約入金期日から[X]日超過。
要件：
- friendlyだけど緊迫感あり
- 良好な協力関係 упоминать
- 尽快対応を希望を明確に
- 連絡先の記載
```

### シーン2：部門間連携

```
【部門間連携】メール帮我写。
背景：我是[部門]、[他の部門]に[具体タスク]のサポートが必要。
要件：
- タスクの重要性と緊迫性を説明
- 必要なサポートを明確に
- 明確な期限を提示
- 命令口吻を避ける
```

### シーン3：上司への報告

```
【業務報告】メール帮我写。
背景：[プロジェクト/タスク]已完成[進捗]、[問題/困難]に遭遇。
要件：
- 結論を先に、详细内容後
- 关键成果とデータを強調
- 問題と解决方案を誠実に説明
- 次stepsで締めくくり
```

### シーン4：断り/辞退メール

```
【丁重なお断り】メール帮我写。
场景：[具体情况]。
要件：
- 感謝または理解を先に表达
- 協力できない理由を説明
- 代替案があれば提供
- 友好的だが態度明確

## AIメール作成の注意点

### ❌ AI原文をそのまま使わない

AIが書いたメールは「标准答案」、あなたのケースは「具体情况」。
 반드시：
- 具体的なプロジェクト名、日付、数字を替换
- 个人风格とトーンを追加
- 不正確な情報がないかチェック

### ❌ 敏感な内容はAIに任せない

機密、纠纷、投诉、辞职这类敏感邮件、AIには無理。
この種のメールは个人で責任を負う、自分で書いた方が安全。

### ❌ フォーマットを忘れるな

AI生成のメールはフォーマットに問題がある可能性：
- 收件人、抄送是否正确
- 附件真的附上了か
- 署名档是否更新

## 上級テク：AIメールアシスタントを訓練

頻出メールシナリオをまとめて、AIに自分の风格を学習させる：

```
私のメール作成偏好：
1. 冒頭は「您好」より「你好」好き
2. 段落は3〜4行に控制在
3. 常用表現：[好きな表現を収集]
4. 結尾通常是「期待您的回复」
```

これをAIにfeedすれば、今後もっと自分らしいメールになる。
