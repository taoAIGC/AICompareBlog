## 三个AI的真实使用感受

### 1. ChatGPT（GPT-4）

**입구**: chat.openai.com

**내 경험**:
- 익숙한 맛을 지닌 최초의 AI 사용
- 빠른 응답과 상쾌한 인터페이스
- 플러그인 생태계가 풍부하고 모든 것이 연결될 수 있습니다.
- 4o 버전은 그림 이해력이 강합니다.
- 무료 버전이면 충분하며, 플러스 버전은 월 20달러입니다.

**이점**:
- 전반적인 능력이 뛰어나고 눈에 띄는 단점이 없습니다.
- 가장 완벽한 플러그인과 생태계
- 신뢰할 수 있는 온라인 검색
- 강력한 다중 모드 기능

**결점**:
- 중국어 답변은 때때로 너무 "번역적"입니다.
- 가끔 "말도 안 되는 소리를 진지하게 한다"
- 국내 접속이 불안정합니다.
- 값비싼

**적절한 장면**:
- 영어 작문 및 번역
- 창의적인 콘텐츠 생성
- 온라인에서 정보를 검색해야 하는 Q&A
- 복잡한 문제를 브레인스토밍

**가격**: 무료 버전이면 충분하며, 플러스 버전은 월 20달러입니다.

---

### 2. Claude 3.5

**입구**: clude.ai

**내 경험**:
- 코딩 실력이 정말 좋고, 특히 Python과 JavaScript 작성이 편합니다.
- 문맥을 이해하는 능력이 뛰어나고, 긴 글을 늘어뜨리지 않고 처리하는 능력이 뛰어납니다.
- 깔끔한 인터페이스, 광고 없음
- GPT보다 거부율이 적습니다.
- 무료 버전에는 하루에 20개의 메시지가 있는데, 이 정도면 충분하지만 충분히 흥미롭지는 않습니다.

**이점**:
- 최강의 코딩능력 (개인 테스트)
- 긴 텍스트 처리 우수
- 답변 스타일은 자연스럽고 로봇처럼 들리지 않습니다.
- Anthropic은 AI에 대해 엄격한 접근 방식을 취합니다.

**결점**:
- 무료 버전에는 크레딧이 너무 적습니다.
- 온라인 검색을 지원하지 않습니다.
- 국내 접속도 안정적이지 못함
- 플러그인 생태계 없음

**적절한 장면**:
- 프로그래밍 개발(강력히 권장)
- 긴 글 작성 및 편집
- 깊은 사고가 필요한 분석
- 번역 작업

**가격**: 무료 버전은 하루 20개 항목, Pro 버전은 월 $20

---

### 3. DeepSeek V3

**입구**: chat.deepseek.com

**내 경험**:
- 국산 AI의 놀라운 성과
- 중국어를 매우 잘 이해하고, 번역의 느낌이 없습니다.
- 무료 할당량은 넉넉하며 기본적으로 무제한입니다.
- 빠른 응답
- 추리력이 대폭 향상되었습니다.
- 관건은 - 국내 접속이 안정적이라는 점!

**이점**:
- 우수한 중국어 능력
- 완전 무료(현재로서는)
- 국내접속이 안정적이다
- 저렴함(API가 GPT보다 저렴함)
- 추론 모델 R1은 무료로 제공됩니다.

**결점**:
- 영어 작문은 여전히 ​​약간 칭글리쉬입니다.
- 창의성 측면에서는 GPT에 비해 약간 약함
- 생태학은 아직 건설 중입니다.

**적절한 장면**:
- 중국어 글쓰기와 회화
- 일일 Q&A 및 업무지원
- 프로그래밍 입문 및 학습
- 비용 절감을 위한 첫 번째 선택

**가격**: 무료! API 가격도 매우 낮습니다.

## 我的真实选择

### 日常工作流

**코드 작성 → Claude**
상대가 없습니다. Claude는 코드 디버깅, 코드 해석 및 코드 최적화를 잘 이해하고 있습니다.

**기사 작성/콘텐츠 제작 → GPT-4**
창의적인 아이디어가 많고 부드러운 표현력이 뛰어나 '아름답게 쓰여진 글'이 요구되는 장면에 적합합니다.

**비용 절감/일상 사용 → DeepSeek**
비용도 들지 않고 나라도 안정됩니다. 어떤 종류의 자전거를 원하시나요?

## 进阶用法：同时对比三个AI

너무 많이 말했지만 실제로 가장 즐거운 사용 방법은 다음과 같습니다.

**AI 비교 플러그인을 사용하면 한 번의 클릭으로 세 명의 AI에게 동시에 질문하여 누가 더 잘 대답하는지 확인할 수 있습니다! **

특정 작업:
1. AI 비교 플러그인 설치
2. 질문을 입력하세요
3. GPT, Claude, DeepSeek에 대한 답변을 한 번에 확인하세요
4. 한 화면에서 비교하고 마음에 드는 것을 선택하세요

이렇게 하면 앞뒤로 전환할 필요가 없어 효율성이 극대화됩니다.

## 写在最后

세 가지 AI를 모두 사용한 후의 느낌은 다음과 같습니다. **완벽한 AI는 없고 적합한 시나리오만 있습니다. **

- 궁극의 코딩능력 추구 → 클로드
- 창의성과 종합역량 추구 → GPT-4
- 경제성과 안정성 추구 → DeepSeek

내 제안은: 세 가지를 모두 등록하고 다양한 시나리오에 대해 서로 다른 AI를 사용하는 것입니다. 결국--
