# AI帮我写简历后，面试机会翻倍了

---

## 1

我表弟小张去年找工作，投了30多份简历，全部石沉大海。

他来找我："哥帮我看看简历是不是有问题？"

我打开一看，好家伙。
"负责公司相关工作"——什么公司？什么工作？
"具备较强沟通能力"——多强？举过例子吗？
"熟悉办公软件"——Word还是Excel？熟练程度？

我说："你这是写简历还是写谜语？HR看了都懵。"

他说："我实在不知道咋写，大学也没干啥。"

我说："没事，我让AI帮你。"

## 2

很多人写简历有两个极端：

要么觉得自己没啥可写，干巴巴几条，像记流水账。
要么写得又长又密，HR看3秒就想关掉。

其实**好的简历 = 经历 + 成果 + 数据**。

没经历？大学作业、社团活动、兼职都行。
没成果？写成"负责XX"不如写成"完成XX，提升XX"。没数据？试试"提高30%"比"效果很好"更有说服力。

具体怎么写？AI能帮你。

## 3

### 简历怎么让AI写

**第一步：把自己的经历倒出来**

别管格式，别管文笔，把你干过的事全写出来：

"大学在学生会宣传部干过，负责做海报、写公众号推文。"
"暑假在电子厂打过工，做流水线。"
"帮邻居小孩补过数学课。"
"自己开过淘宝店，卖过游戏装备。"

**第二步：让AI帮你改写成简历语言**

提示词这样写：

```
你是一个有10年经验的HR，现在帮我优化简历。
我是一名应届生，以下是我的经历：
[粘贴你的经历]

请帮我改写成简历格式，要求：
1. 每条经历包含"做了什么"+"结果如何"
2. 使用动词开头
3. 突出可量化的成果
4. 控制在一页纸以内
```

**第三步：针对不同岗位微调**

投技术岗？突出代码、项目、解决问题的能力。
投运营岗？突出策划、数据分析、用户沟通的经验。
投销售岗？突出沟通能力、销售业绩、与人打交道的故事。

一个模板打天下，不如针对性的改一改。

## 4

### 面试问题怎么让AI准备

简历写完，面试才是硬仗。

很多人面试前不知道准备啥，要么干等着紧张，要么背一堆废话。

其实**面试准备 = 常见问题 + 你的故事 + 模拟练习**。

**常见问题AI帮你列：**

```
你是一个资深面试官，帮我预测这个岗位的面试问题：
岗位：新媒体运营
要求：负责公众号、微博内容策划，粉丝增长

请列出10个最可能被问到的问题，并给出回答思路。
```

拿到问题后挨个准备，把你的故事套进去。

**你的故事怎么讲：**

STAR法则了解一下：
- **S**ituation（情境）：当时什么情况？
- **T**ask（任务）：你的目标是什么？
- **A**ction（行动）：你具体做了什么？
- **R**esult（结果）：结果怎么样？

比如被问到"你觉得你沟通能力怎么样"：

"（S）大学时我发现社团活动报名人数很少。（T）我想提高到100人。（A）我做了三件事：1）设计更吸引人的海报；2）逐个联系社团负责人；3）在校园群发通知。（R）最终报名人数达到156人，比之前多了60%。"

有情境有行动有数据，比干巴巴说"我沟通能力强"有力多了。

## 5

### 面试紧张怎么办

我表弟面试前紧张得睡不着。

我说："你紧张啥？大不了就是不过呗。"

他说："就是害怕嘛。"

我说："教你个办法，面试前把AI当成面试官，先练三遍。"

**模拟面试提示词：**

```
你是一个面试官，面试岗位是[岗位名]。
请向我提出5个常见面试问题，每次只问一个，等我回答后你再点评，然后问下一个。
态度可以严格一点，指出我回答的问题。
```

跟AI练一遍，心里就有底了。真正面试的时候，发现"这些题我练过"，紧张感少一半。

## 6

### 谈薪资怎么让AI教

薪资是面试最难开口的部分。

低了亏自己，高了怕黄掉。

**让AI帮你做薪资调研：**

```
我要面试[城市]的[岗位]，请问这个岗位的平均薪资是多少？
请给出不同经验级别的薪资范围。
```

心里有个数之后，怎么开价也有底气。

**让AI帮你模拟谈薪资对话：**

```
我面试的公司给我开了8000一个月，我想要10000。
请模拟一段谈薪资的对话，教我怎么开口，怎么说服对方。
```

提前练练，真的到了那个场合不至于说不出话。

## 7

### 面试后怎么跟进

面完试，很多人就是干等着。

其实**面试后跟进 = 感谢消息 + 复盘总结**。

**感谢消息怎么写：**

```
我今天参加了[公司]的[岗位]面试，面试官是[名字]。
请帮我写一段100字以内的感谢消息，表达我对这个岗位的热情，以及期待回复。
```

**复盘怎么写：**

每次面试完，花10分钟记下来：
- 哪些问题回答得好？
- 哪些问题回答得烂？
- 下次怎么改进？

我表弟按这个方法练了一个月，后面几场面试一个比一个顺。

## 8

### 最后说几点

**第一，简历是敲门砖，不是你的全部。**

写得好不代表能力强，但写得烂可能连面试机会都没有。花点时间打磨不亏。

**第二，AI是辅助，不是替代。**

AI帮你整理、帮你优化、帮你练习，但最终上场的是你自己。该记的要记，该练的要练。

**第三，面不上很正常，别灰心。**

找工作是个概率事件，面10家成1家就赚了。每次面试都是学习机会，复盘多了，成功的概率就上去了。
