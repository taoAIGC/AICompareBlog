## Tipos Comuns de Erros de Código

### Tipo 1: Erros de Sintaxe
Mais comuns, mas o IDE geralmente destaca em vermelho:
- Erros de ortografia
- Ponto e vírgula/parênteses faltando
- Problemas de indentação

### Tipo 2: Erros de Execução
O código roda, mas falha em alguma linha:
- Exceções de ponteiro nulo
- Índice de array fora do intervalo
- Falhas de conversão de tipo

### Tipo 3: Erros Lógicos
O código roda sem erros, mas os resultados estão errados:
- Condições de loop escritas incorretamente
- Lógica de julgamento invertida
- Algoritmo escrito incorretamente

### Tipo 4: Problemas de Desempenho
O código roda, mas é extremamente lento:
- Loops infinitos
- Consultas N+1
- Vazamentos de memória
