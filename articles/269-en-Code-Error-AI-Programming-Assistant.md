## Advanced Tips

### Tip 1: Combined Use
My combo: Copilot for writing + Claude for learning + DeepSeek for emergency.

### Tip 2: Follow-up Questions
If AI answer isn't satisfactory, ask follow-up:
> "Why does this problem occur? Is there a better solution?"

### Tip 3: Specify Role
When asking, specify role for more professional answers:
> "You are a senior backend engineer, please help analyze this code's performance issues."
