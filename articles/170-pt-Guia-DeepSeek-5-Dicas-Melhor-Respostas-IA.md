##Dica 2: Como escrever a palavra prompt

### Não use por muito tempo

DeepSeek não é o mesmo que ChatGPT.

ChatGPT escreve uma longa lista de palavras imediatas e lê todas elas.

Se você escrever DeepSeek por muito tempo, ele sairá facilmente do caminho.

**Escrita correta**:
```
Escreva um artigo sobre comunicação no local de trabalho, com cerca de 1.000 palavras. O público-alvo são os recém-chegados que acabaram de ingressar no trabalho. Deve ser prático e não muito teórico.
```

**Maneira errada de escrever**:
```
Você é um especialista sênior em recursos humanos com 10 anos de experiência em recrutamento e é bom em lidar com questões de relacionamento no local de trabalho. Por favor, escreva um artigo de acordo com os seguintes requisitos: primeiro... em segundo lugar... então... finalmente...
```

É muito longo, DeepSeek leu o final e esqueceu o começo.

### Pergunte passo a passo

Faça perguntas complexas à parte.

Por exemplo, se você quiser escrever um artigo:

Etapa um:
```
Ajude-me a fazer um esboço do artigo "Habilidades de comunicação no local de trabalho", 5 partes
```

Depois que DeepSeek deu o esboço:

Etapa dois:
```
Com base no primeiro esboço, escreva um parágrafo detalhado de 200 palavras
```

Etapa três:
```
Continue escrevendo a segunda parte do esboço
```

Isso funciona muito melhor do que fazer todas as perguntas de uma vez.

### Limpar formato

Às vezes, o DeepSeek pode exagerar e escrever um monte de coisas que você não precisa.

Diga exatamente qual formato você deseja.

```
Por favor, exiba-o em forma de tabela. Cada item de comparação deve incluir: função, desempenho do DeepSeek, desempenho do ChatGPT
```

```
Explique em pontos, no máximo 5 itens, cada item não deve ultrapassar 50 palavras.
```

Não será grátis para jogar.

##Dica 3: uso baseado em cenário

### Escrever cópia

DeepSeek é muito bom em escrever direitos autorais chineses.

Especialmente o tipo realista.

Por exemplo, escrever uma cópia para Moments:
```
Ajude-me a escrever um círculo de amigos, dizendo que meus colegas me deram chá com leite e bolo no chá da tarde desta tarde. Seja um pouco engraçado.
```

DeepSeek lhe dará uma versão interessante.

### Escrever código

Também pode ser usado ao escrever código.

Mas o efeito real medido não é tão bom quanto o de Claude.

A escrita do código de Claude é mais padronizada e seus comentários são mais claros.

O DeepSeek às vezes apresenta pequenos problemas ao escrever código, por isso é recomendável verificá-lo duas vezes.

###Escreva um artigo

DeepSeek pode escrever artigos de contas oficiais, respostas de Zhihu e direitos autorais de Xiaohongshu.

Especialmente para conteúdo que requer "senso de internet", o senso de linguagem do DeepSeek é muito bom.

### traduzir

A tradução do DeepSeek é mais natural que o Google Translate.

Não é uma tradução automática rígida, mas parece fala humana.

```
Por favor, ajude-me a traduzir este parágrafo do inglês para o chinês. Deve ser natural e não literal.
```

Funciona muito bem.

### Brainstorming

Não consegue pensar em um tema para escolher? Não sabe como tomar uma decisão?

Deixe o DeepSeek ajudá-lo a fazer um brainstorming.

```
Eu sou um blogueiro no local de trabalho. Escrevo principalmente sobre crescimento no local de trabalho e habilidades de comunicação. Por favor, me dê 10 tópicos para escolher.
```

Isso lhe dará um monte de ideias.

## Dica 4: Guia para evitar armadilhas

### Não confie muito

DeepSeek às vezes fala bobagens sérias.

Ele fabrica dados e cotações para parecer profissional, mas na verdade é falso.

**Solução**:
- Verificação manual de informações importantes
- Envolve verificação multi-ângulo do conhecimento profissional
- Não copie diretamente para o documento oficial

### Questões sensíveis

DeepSeek possui revisão de conteúdo e há algumas perguntas que ele não responde.

Por exemplo:
- Como ignorar o pagamento
- Como quebrar software
- Eventos históricos sensíveis

Ele rejeitará essas perguntas ou lhe dará um discurso oficial.

### Restrições contextuais

A janela de contexto do DeepSeek não é tão longa quanto a do Claude.

Ao final de escrever um longo artigo, você pode esquecer o que foi dito anteriormente.

**Solução**:
- Escreva em seções, cada seção deve ter no máximo 2.000 palavras.
- Repita informações importantes no início de cada parágrafo
- Resuma manualmente o conteúdo anterior e alimente-o, se necessário

### Não pergunte sobre coisas que são muito novas.

Os dados de treinamento do DeepSeek têm data de validade.

Pode não saber o que acontecerá depois de 2024.

Pergunte "O que aconteceu recentemente" e ele lhe dará uma resposta desatualizada.

##Dica 5: uso avançado

### Palavra de prompt do sistema

DeepSeek oferece suporte a palavras de prompt do sistema.

Clique em Configurações no canto superior direito para definir o comportamento padrão da IA.

Por exemplo:
```
Você é um consultor de negócios profissional. Responda às perguntas de forma concisa e eficaz e use dados e casos para apoiar suas opiniões.
```

Uma vez definido, todas as conversas subsequentes terão efeito.

### Modelo de conversa

Palavras úteis de prompt podem ser salvas como modelos.

Atualmente, o DeepSeek não possui um recurso de modelo oficial, mas você pode:

1. Escreva em um documento
2. Copie e cole quando necessário
3. Ou faça uma captura de tela e salve-a

### chamadas de API

Os desenvolvedores podem acessar via API.

A API do DeepSeek é mais barata que o OpenAI.

A documentação é escrita de forma clara e amigável ao desenvolvedor.

### Implantação local

Código aberto significa que você pode baixar o modelo e executá-lo localmente.

Requisitos de configuração:
- Placa gráfica com pelo menos 16 GB de memória de vídeo
- Memória 32GB ou mais
- Espaço em disco rígido 100GB+

Computadores comuns não conseguem executá-lo, por isso é recomendado o uso de serviços em nuvem.
