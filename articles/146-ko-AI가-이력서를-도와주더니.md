# AI가 이력서를 도와주더니 면접 기회가 2배가 되었어요

---

## 1

사촌동생 샤오장이去年 취직 준비했어요. 30통 이상의 이력서를 보냈지만 하나도 답변이 없었어요.

나한테 와서 말했어요. "형, 이력서 문제 있는지 봐줘."

열어봤는데. 와.
"회사 관련 업무 담당" — 어느 회사? 어떤 업무?
"沟通能力强" — 얼마나 강해? 예시 있니?
"오피스 소프트웨어 사용 가능" — Word 아니면 Excel? 어느 정도?

말했어요. "이력서 쓰는 거냐 수수께끼냐? 인사담당者们会困惑한다."

말했어요. "정말不知道怎么寫. 대학에서 별로 안 했는데."

말했어요. "�찮아, AI한테 도와달라 해."

---

## 2

대부분의 사람들은 두 가지 극단적인 방식으로 이력서를 써.

쓸 게 없다고 생각해서 그냥 리스트만 쓰거나.
너무 많이 써서浓密해서, 인사담당자 3초만에 닫고 싶어하게 돼.

실제로 **좋은 이력서 = 경험 + 성과 + 숫자**.

경험 없으면? 대학 과제, 동아리 활동, 아르바이트 다 돼.
성과 없으면? "X 담당"은 "X 완료, X를 X% 개선"보다 약해.
숫자 없으면? "30% 개선"이 "좋은 결과"보다 효과적.

어떻게 써? AI가 도와줄 수 있어.

## 3

### AI에게 이력서 쓰게 하는 방법

**1단계: 모든 경험 적어**

형식 신경 쓰지 말고, 한 거 다 적어:

"학생회 홍보부서에서 포스터 제작하고 공식 계정 글쓰기 담당했어요."
"여름에 전자공장打工했어요, 라인 작업했구요."
"이웃집 아이한테 수학 과외했어요."
"타오바오에서网店 운영했어요, 게임 장비 판매했어요."

**2단계: AI에게 이력서 언어로 다시 쓰라고 하기:**

```
10년 경력의 인사담당자입니다. 이력서 최적화 도와주세요.
저는 신입毕业生입니다.以下是경험입니다:
[경험 붙여넣기]

이력서 형식으로 다시 쓰기, 요구사항:
1. 각 항목은 "무엇을 했는지" + "결과" 포함
2. 동사로 시작
3. 정량화 가능한 성과 강조
4. 한 페이지로局限
```

**3단계: 직종에 맞게微調整**

기술 직종 지원? 프로그래밍, 프로젝트, 문제 해결 능력 강조.
운영 직종 지원? 기획, 데이터 분석, 사용자 커뮤니케이션 강조.
영업 직종 지원? 커뮤니케이션 능력, 매출 실적, 사람과 일하기 강조.

전부 같은 템플릿보다 맞춤형 수정이 더 좋지.

## 4

### AI로 면접 준비하는 방법

이력서는 썼어. 면접이 본격적인 시험이야.

많은 사람들이 면접에 뭘 준비해야 할지 몰라. Either 기다리면서 긴장하거나,废水 다 외워.

실제로 **면접 준비 = 흔한 질문 + 내 이야기 + 연습**.

**AI가 흔한 질문リスト作成:**

```
경험 많은 면접관입니다. 이 직종의 면접 예상 질문 도와주세요:
직종: 신규 미디어 운영
요구사항: 공식 계정, 위보 콘텐츠 기획, 팔로워 성장

가장 있을 법한 10개 질문과 답변思路 제시.
```

질문 받으면 하나씩 준비해. 내 이야기 넣기.

**내 이야기 어떻게 말해:**

STAR 방법 알아:
- **S**ituação (상황): 상황이 어땠어?
- **T**arefa (과제): 목표가 뭐였어?
- **A**ção (행동): 구체적으로 뭘 했어?
- **R**esultado (결과): 결과가 어땠어?

예를 들어 "커뮤니케이션 능력 좋다고 생각하세요?"라는 질문에:

"(S) 대학 때 동아리 활동 신청人数 너무 적었어. (T) 100명까지 늘리고 싶었어. (A) 세 가지 했어: 1) 더 끌리는 포스터 설계; 2) 동아리负责人 하나씩 연락; 3) 학교 그룹에 알림. (R) 결국 156명 신청 받아, 이전보다 60% 증가."

상황+행동+결과 있으면 "저 커뮤니케이션 강해요"보다 훨씬 좋지.

## 5

### 면접 전 긴장되면 어때

샤오장이 면접 전 잠을 못 잘 정도로 긴장했어.

말했어. "뭐 그렇게 긴장해. 안 되도 괜찮잖아."

말했어. "그냥 무서울 뿐이에요."

말했어. "방법 알려줄게. 면접 전 AI를 면접관으로 만들고 세 번 연습해."

**모의 면접 프롬프트:**

```
[직종] 면접관입니다.
5개 흔한 면접 질문给我. 한 번에 하나씩问我, 대답하면 평가하고 다음 질문.
엄격하게 해도 되고, 문제 있으면 지적해도 돼.
```

AI랑 한 번 연습하면 자신감 생겨. 실제 면접 때 "이거 연습했네" 하고 보면 긴장 반 줄어.

## 6

### AI帮助下薪资谈判

연봉은 면접에서 제일 꺼내기 어려운 부분이야.

낮으면 내가 손해, 높으면 탈alu可能.

**AI한테 연봉 조사하게:**

```
[도심]에서 [직종] 면접 봐요. 이 직종 평균 연봉이야?
경력 단계별 연봉 범위 알려줘.
```

숫자 알면 근거 생겨.

**AI한테 연봉 협상 대화 시뮬레이션하게:**

```
회사가 한 달 8000 줬는데, 10000 받고 싶어.
연봉 협상 대화 simulate 해줘, 어떻게 꺼내고 어떻게说服하는지 알려줘.
```

미리 연습해. 그때 되면 말도 안 나온다고 하면 안 되잖아.

## 7

### 면접 후 팔로업

면접 보고 그냥 기다리는 사람很多.

실제로 **팔로업 = 감사 메시지 + 복기**.

**감사 메시지 어떻게 써:**

```
오늘 [회사]에서 [직종] 면접 봤어요. 면접관은 [이름]이었어요.
100자 이내 감사 메시지 써줘. 이 직종에 대한 열정과 답변 기다린다는 내용.
```

**복기 어떻게 써:**

매번 면접 보고 10분 걸려서 기록:
- 어떤 질문 잘 봤어?
- 어떤 질문 못 봤어?
- 다음엔 어떻게改进해?

샤오장이 이 방법 한 달 연습했더니, 다음 면접마다 더 좋았어.

## 8

### 마지막으로 몇 마디

**첫째, 이력서는敲门砖, 전부가 아니야.**

잘 쓴다고 다 능력 있는 건 아니지만, 못 쓰면 면접 기회도 없어. 시간 들여 다듬을 가치 있어.

**둘째, AI는 도우미, 대리가 아니야.**

AI가 정리하고 최적화하고 연습하도록 도와주지만, 실제로 하는 건 너自身. 외울 건 외우고, 연습할 건 연습해야 해.

**셋째, 면접 안 되는 거 정상이에요.泄气 말아요.**

求职는 확률的游戏. 10군데 지원해서 1군데 되면_ok.每次 면접은 학습 기회. 복기 많이 하면 성공 확률 올라가.
