## 서론

 AI 도구가 너무 많아!

DeepSeek, Kimi, Doubao, Tongyi Qianwen, ChatGPT, Claude...... 모두 자기가 제일 좋다고 주장한다.

반년 동안這些 주요 AI들을 다 사용해본 사람として 실측 비교를 보여줄게. 허드렛말 없이 **자신에게 맞는 AI 도구**가 뭔지 알려줄게.

## 주요 AI 도구 비교

### 1. DeepSeek

**접속**: chat.deepseek.com

**실제 사용 후기**:
- 2024년 국산 AI 중 최대 반전
- 중국어 능력 최상위권, 번역 느낌 없음
- 무료 할당량 거의 무제한
- 추론 모델 R1 무료这次赢大了

**장점**:
- 중국어 능력 최상위권
- 완전 무료
- 중국에서 접속 안정적
- API 가격 매우 저렴

**단점**:
- 영어作文이 Chinglishっぽくなる
- 창의성이 조금 약함

**적합한 사람**:
- 중국어 Q&A 및 글쓰기
- 프로그래밍 질문
- 절약派的 사람

---

### 2. Kimi

**접속**: kimi.moonshot.cn

**실제 사용 후기**:
- Moonshot社製、정말 좋음
- 长文处理能力变态、200만자 지원
- 무료 할당량 넉넉함
- 광고 없는 깨끗한 인터페이스

**장점**:
- 长文 이해能力强
- 중국어 표현 자연스러움
- 파일 업로드 및 읽기 지원
- 응답速度快

**단점**:
- 가끔 "뻗음"
- 창의성은 보통

**적합한 사람**:
- 문서 요약 및 Q&A
- 긴 글 작성
- 자료 정리

---

### 3. Doubao

**접속**: www.doubao.com

**실제 사용 후기**:
- ByteDance社製
- 응답 속도 빵빵
-简洁하고 사용하기easy
- 일상 사용에는 무료 할당량 충분
- 관련 도구 많음

**장점**:
- 응답 속도 최상
- 생태계풍부
- 안정적이고 신뢰할 수 있음
- 초보자向き

**단점**:
- 깊이 있는 능력은 조금 약함
- 창의성 보통

**적합한 사람**:
- 일상 Q&A
- 간단한 글쓰기
- 빠른 정보 획득

---

### 4. 통의천문

**접속**: tongyi.aliyun.com

**실제 사용 후기**:

**장점**:

**단점**:

**적합한 사람**:
- 논리적 질문
- 프로그래밍 도움말
- 일상 오피스 워크

## 상황별 추천: 뭘 써야 할까?

### 일상 Q&A → Doubao
빠름, 뭐든 답해줌, 기다릴 필요 없음

### 글쓰기 → DeepSeek
중국어 표현 자연스럽고 무료 할당량 많음

### 문서 읽기 → Kimi
긴 텍스트 넣으면 자동으로 요약해줌

### 프로그래밍 → 통의천문
논리 명확하고 코드 품질 좋음

### 절약 → DeepSeek
무료 할당량 제일 많음

## 상급用法: 여러 AI 동시에 쓰기

사실 가장爽的做法은:

**AI比一比 플러그인 쓰면, 한번에 여러 AI를 열어서 누가 더 좋은 답을 주는지 확인할 수 있음!**

사용법:
1. AI比一比 플러그인 설치
2. 아이콘 클릭
3. DeepSeek, Kimi, Doubao 한번에 열기
4. 제일 좋은 답 쓴 거 쓰기

이렇게 하면换来换去할 필요 없음 - 하나의 질문에複数の 답 중 제일 좋은 거 선택.
