## AI 데이터 분석으로 무엇을 할 수 있나요

### 1. 데이터 정리
정돈되지 않은 데이터 AI에게 주면, 중복 제거, 결측치填补, 형식 표준화를 자동으로 해줍니다.手動으로 수정할 필요 없음.

### 2. 데이터 분석·해석
데이터가 무슨 의미인지 모르겠어요? AI가 트렌드 분석, 규칙 발견, 인사이트 정리帮我理解数据在说什么.

### 3. 자동 차트 생성
AI에게 차트 유형告诉我, 자동으로 예쁜 색상으로 생성해줍니다. 하나씩 만들 필요 없음.

### 4. 수식 작성
복잡한 Excel 함수 기억 못함? AI에게 무엇을 원하는지 말하면, 수식 작성하고 원리도 설명해줍니다.

### 5. 데이터 보고서 작성
분석한 데이터를 보고서로 못 만드나요? AI가 깔끔한 텍스트로 정리하고 논리 구성帮我.

## 일반적인 데이터 분석 시나리오

### 시나리오 1: 판매 데이터 분석

**단계**:
1. 판매 데이터를 테이블로 정리 (Excel 또는 CSV)
2. AI에게 이 프롬프트:

```
이 판매 데이터 분석해주세요:
[데이터 붙여넣기]

분석해주세요:
1. 전체 판매 트렌드 상승? 하락?
2. 어떤 상품이 잘 팔림?
3. 어느时间段 판매 가장 높음?
4. 주목할 만한 이상 있나요?
5. 데이터 기반 판매 제안 3개
```

---

### 시나리오 2: 사용자 데이터 분석

**단계**:
1. 사용자 데이터 정리 (출처, 나이, 위치, 활성도 등)
2. AI에게 보내기:

```
이 사용자 데이터 분석해주세요:
[데이터 붙여넣기]

분석해주세요:
1. 사용자가 주로 어디서 왔나요?
2. 나이/직업 분포는?
3. 사용자 활성도는? 누구最强?
4. 사용자 유지율은?
5. 사용자 활성度を提升하는 방법은?
```

---

### 시나리오 3: 재무 보고서 분석

**단계**:
1. 재무 보고서 데이터 준비
2. AI에게 해석 도와달라고:

```
이 재무 데이터 분석해주세요:
[데이터 붙여넣기]

분석해주세요:
1. 매출 상황은?
2. 주요 비용은 어디?
3. 이익률은?
4. 전년/분기 대비 변화는?
5. 문제점 및 개선 제안은?
```

## 중요 참고사항

### ❌这样做

1. **민감한 데이터 직접 업로드하지 마세요** - AI에게 주기 전에 민감한 데이터 익명화
2. **AI 판단에 완전히 의존하지 마세요** - AI 분석은 참고용, 중요한 판단은 사람이
3. **데이터 정확성 무시하지 마세요** - AI가 데이터를 잘못 이해할 수 있으니 반드시 확인
4. **비즈니스 맥락 무시하지 마세요** - 분석은 비즈니스 고려해야, AI는 당신具体情况몰라

### ✅这样做

1. **미리 데이터 정리하세요** - 데이터整洁 = 분석 정확
2. **분석 목표 명확하게 하세요** - AI에게 무엇을 해결하고 싶은지 말하면 결과更好
3. **결과 반드시 검증하세요** - AI 분석 다시 확인해서 정확성 보장
4. **비즈니스 사고와 결합하세요** - AI 분석과 당신의 비즈니스 경험 결합
