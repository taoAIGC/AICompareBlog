# 进阶技巧

## 技巧1：组合使用才是王炸

不同场景用不同工具：

- 英文视频 → ElevenLabs（音质好）
- 中文视频 → 讯飞（中文自然）
- 大量生产 → Azure（稳定）
- 预算有限 → TTSMaker（免费）
- 深度定制 → GPT-SoVITS（开源）

## 技巧2：调整语速和语调

大多数工具都支持参数调整：

- 语速：正常语速的0.8-1.2倍
- 语调：根据内容调整，讲解用平稳，故事用起伏
- 停顿：适当添加停顿，让AI读得更自然

## 技巧3：后期处理提升质量

原始AI语音加一点后期，效果提升明显：

- 降噪：去除底噪
- 均衡：调整频率
- 压缩：统一音量
- 添加背景音乐：增加氛围感

## 技巧4：多语言内容找专业工具

每种语言都有"王者"：

- 英文 → ElevenLabs
- 中文 → 讯飞
- 日文 → CoeFont或Voicevox
- 德法西 → Azure TTS
