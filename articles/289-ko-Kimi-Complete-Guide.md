## Kimi란?

Kimi는 Moonshot AI가 개발한 AI 어시스턴트로, 200만 자 컨텍스트를 지원해서 유명합니다 - 무료 版으로도 쓸 수 있어서 AI 업계에서 독특한 존재예요.

간단히 말하면, Kimi는 다음을 도와줄 수 있어요:
- 아주 긴 문서를 읽기
- 다양한 콘텐츠 쓰기
- 질문에 답하기
- 정보 정리하기

지능형 어시스턴트예요.

## 등록하고 사용하는 방법은?

### 1단계: 등록

1. Kimi 웹사이트 방문: kimi.moonshot.cn
2. "시작하기" 클릭
3. 전화번호로 등록, SMS로 인증
4. 인증 필요 없음, 바로 사용 가능

### 2단계: 채팅 시작

등록 후 입력 상자에 글씨 치면 됨. 지원:
- 텍스트 입력
- 파일 업로드 (PDF, Word, TXT, 이미지 등)
- 링크 보내기

## Kimi는 뭘 할 수 있을까?

### 기능 1: 긴 문서 읽기 (핵심 기능)

**사용 시기**:
- 논문 읽기
- 보고서 읽기
- 계약서 읽기
- 동료의 긴 문서 읽기

**사용 방법**:
파일을 입력 상자로 드래그하거나, 복사해서 붙여넣고, 알고 싶은 것 물어보면 됨.

**실측 결과**:
- 50페이지 PDF, 10초 요약
- 한 번에 50개 파일 업로드 가능
- 200만 자 컨텍스트는超长内容도 기억할 수 있다는 뜻

**질문 예시**:
- "이 논문의 주요 포인트 요약해줘"
- "이 계약서에서 주의할 조항은?"
- "이 파일들의 유사점과 차이점 비교해줘"

##新手常见问题

### Q1: 무료 版够用?
A: 완전히够用. 무료 版은 200만 자 컨텍스트 지원해서 대부분의 문서에 충분. 유료 版은 할당량이 더 많음.

### Q2: 파일 크기 제한 있어?
A: 파일당 100MB 이하, 한 번에 50개까지. 일반 사용자에겐 충분.

### Q3:响应慢怎么办?
A: 피크 시간엔 느릴 수 있음. 아침/저녁 피크 시간 피하면 빨라짐.

### Q4:回答不准怎么办?
A: 더 많은 맥락 제공하며 다시 물어보기. 질문이 구체적일수록 답변이 정확해짐.

## 使用技巧

### 技巧1: AI에게 역할 부여

**错误**: "뭔가書いて"
**正确**: "10년 경력 HR이야, 이력서 최적화 도와줘"

### 技巧2: 分步处理

**错误**: "이 회사 문제全部 분석해줘"
**正确**: 먼저 "이 회사 비즈니스 모델이 뭐야?", 그 다음 "경쟁 상황이야?"

### 技巧3: 예시 제공

**错误**: "이 형식으로書いて"
**正确**: "이 형식으로 문장 3개 작성해: [예시 붙여넣기]"

### 技巧4: 형식 지정

**错误**: "요약해줘"
**正确**: "이 3개方案的 장단점을 표로 정리해줘"
