## 写在前面

翻訳ソフトウェアを使用したことがない人はいますか?しかし、AI時代の到来により、翻訳ツール間の格差は以前よりもはるかに広がっています。

昔は「だいたい理解できる」程度でしたが、今は「人訳に近い」状態になっています。 DeepL の突然の台頭により、Google は大きなプレッシャーにさらされています。 GPT-4 は翻訳もでき、それぞれに独自のスキルがあります。

学術論文の翻訳に適したもの、日常のチャットの翻訳に適したもの、数十の言語をサポートするもの、十分な無料割り当てがあるもの、法外な料金を請求するもの...

今日は、実際の経験に基づいて、最適なツールを見つけるのに役立ついくつかの主流の AI 翻訳ツールについて説明します。

## 详细对比

### 1. DeepL——翻译质量的"天花板"

DeepL は過去 2 年間の翻訳成果物であり、その評判が非常に高かったため、業界から撤退しました。

**优点**：
- 特にヨーロッパ言語の最高品質の翻訳
- 31の言語をサポート
- 強力な文書翻訳機能（Word、PPT、PDFの直接翻訳）
- 便利なブラウザプラグイン
・用語集機能（用語の統一性を保つ）
- 中国語翻訳は自然でスムーズです
- 無料版も利用可能
- インターネットをサーフィンする必要はありません

**缺点**：
- 無料版には割り当て制限があります (月あたり 5,000 文字)
- 一部のマイナー言語の限定的なサポート
- リアルタイムの会話翻訳はサポートされていません
- 有料版は高価です (月額 10 ユーロから)
- 学術用語は時々不正確です

**适合谁**：需要高质量翻译的用户、跨境电商、留学生、翻译工作者

**我的使用感受**：DeepL是"专业级"选手。

私は文書、契約書、電子メール、Web ページの翻訳に DeepL を使用しましたが、その経験は非常に良好でした。特に英語-中国語の翻訳はGoogle翻訳よりもずっと自然です。

以前英語の論文を翻訳したとき、Google で翻訳されたものを再修正する必要がよくありました。 DeepLは基本的にそのまま利用可能です。これにより本当に時間が節約されます。

最も便利なのは文書翻訳機能です。 Word ファイルをドラッグするだけで、形式は変更されず、翻訳がダウンロードされます。各画像をコピーして貼り付ける必要はありません。

欠点は無料枠が少なすぎることです。 5,000 文字というと多すぎるように思えますが、数本の論文を翻訳するには十分すぎるでしょう。有料版は月額 10 ユーロからで、学生にとってはあまり親切ではありません。

### 2. Google Translate——功能最全的"老大哥"

Google 翻訳は、翻訳分野では最も古い経験と最も完全な機能を備えた古いプレーヤーです。

**优点**：
- 世界最多の 130 以上の言語をサポート
- 完全に無料
- リアルタイム会話翻訳
- 写真翻訳、音声翻訳、手書き翻訳
- オフライン翻訳機能
- ブラウザプラグインとモバイルアプリは優れたエクスペリエンスを提供します
- 登録せずに直接使用できます

**缺点**：
- 中国語-英語翻訳の品質はDeepLほど良くありません
- 学術/専門文書の不正確な翻訳
- 無料版には使用制限があります
- 時々変な翻訳がある
- 疑わしいプライバシーポリシー

**适合谁**：需要多语言支持的用户、日常翻译、紧急翻译场景

**我的使用感受**：Google Translate是"全能型"选手。

Google 翻訳の最大の利点は、対応言語が多いことです。 130 以上の言語。基本的に市場で入手できるすべての言語をカバーします。アイスランド語やスワヒリ語などのマイナー言語は、Google 翻訳でのみサポートされています。

日常の翻訳にとても便利です。携帯電話でアプリを開き、写真を撮ると、メニューと説明書を翻訳できます。海外旅行時にとても便利な機能です。

しかし、翻訳品質を追求するなら、DeepL の方が優れています。私は比較してみました。同じ英語のニュース記事を使用して、Google と DeepL に翻訳させたところ、DeepL の結果はより人間の音声のように読み取れました。

もう一つの問題は、Google 翻訳がしばしばとんでもない翻訳を生成することです。以前、ゲームの説明文を翻訳したことがありますが、「三人称視点」を「第三の神の視点」と訳してしまい、とんでもないことになってしまいました。

### 3. GPT-4——理解能力最强的"全能选手"

GPT-4 はチャットマスターであるだけでなく、一流の翻訳機能も備えています。

**优点**：
- 高い理解力と文脈を理解する能力
- 翻訳スタイルを指定可能（形式的、口語的、学術的）
- 用語説明と追加説明をサポート
- テキストを複数の言語に同時に翻訳できます
- スラングやイディオムを原文で説明できる
- クリエイティブなコンテンツの適切な翻訳

**缺点**：
- API 呼び出しが必要となり、コストが高くなります
- 専門的な翻訳ツールではないため、速度が遅い
- 無料版をそのまま使用することはできません
- 科学的なインターネットへのアクセスが必要です
- 1 つのエントリには文字数制限があります
- 専用の文書翻訳機能なし

**适合谁**：需要创意翻译、复杂语境翻译、术语解释的用户

**我的使用感受**：GPT-4是"智慧型"选手。

GPT-4 の翻訳はコンテキストの理解に重点を置いています。 「これは子供向けのおとぎ話だから、もっと子供っぽく訳してください」と言えば、本当にそれをやってくれます。

かつて、マーケティングコピーを翻訳する必要があったのですが、それには「現実的でオンライン的なセンス」が必要でした。 GPT-4 によって翻訳されたバージョンは、DeepL よりも風味豊かで、対象読者のトーンに沿っています。

原文のジョークを説明してもらうこともできます。ダジャレの多い英語コンテンツを翻訳する場合は、最初に GPT-4 に説明させてから翻訳させます。最終的な効果は直接翻訳よりもはるかに優れています。

欠点は高価なことです。 API 呼び出しごとにコストがかかり、何度も翻訳すると、そのコストは耐えられないものになります。また、速度も遅いため、迅速な結果が必要なシナリオには適していません。

### 4. 腾讯翻译——中文优化的"本地选手"

Tencent Translate は国内翻訳ツールの代表であり、中国語の最適化において優れた成果を上げています。

**优点**：
- 高品質の中国語翻訳
- 無料使用をサポート
- WeChat ミニ プログラムとブラウザ プラグインが利用可能
- リアルタイム翻訳機能
- 国内のインターネットコンテンツの翻訳に適しています
- インターネットをサーフィンする必要はありません
- 素早い対応

**缺点**：
- 少数の言語をサポートします (わずか十数言語)
- 英語以外の言語の平均的な品質
・文書翻訳機能が弱い
・用語集などの機能が少ない
- 国際化が不十分

**适合谁**：需要中英互译的用户、国内用户、日常办公

**我的使用感受**：腾讯翻译是"接地气"选手。

Tencent Translate は、中国語のインターネット用語の翻訳に特に優れています。 「involution」や「lying flat」などの単語の英語翻訳は、Google や DeepL の翻訳よりも正確です。

国内のイベント通知、お知らせ、ソーシャル メディア コンテンツの翻訳には、Tencent Translation の方が良い結果が得られます。 「Double イレブン」は「ダブル イレブン」ではなく「ダブル 11」と翻訳されるべきであることがわかっています。

速度も非常に速く、基本的には数秒以内です。無料の割り当ては基本的に毎日の使用に十分です。

欠点は、言語の種類が少なすぎることです。日本語、韓国語、フランス語、その他の言語を翻訳したい場合はうまく機能しません。さらに、これらの小さな言語の翻訳品質も平均的です。

### 5. 彩云小译——网页翻译的"神器"

Caiyun Xiaoyi は Web ページの翻訳に特化したツールで、プラグインの使い方は非常に簡単です。

**优点**：
- 最高のウェブ翻訳体験
- ブラウザのプラグインがインストールされており、すぐに使用できます
- インテリジェントな二か国語コントロールディスプレイ
- PDF翻訳をサポート
- 無料割り当ては大丈夫です
- インターネットをサーフィンする必要はありません
- 外国語の情報を読むのに適しています

**缺点**：
- 主にWebページ/文書翻訳用
- サポートされる言語が少なくなります
- 人間による翻訳には料金がかかります
・比較的シンプルな機能
- 精度が変動する

**适合谁**：经常阅读外文网站的人、学术研究者、新闻爱好者

**我的使用感受**：彩云小译是"浏览器伴侣"选手。

Caiyun Xiaoyi の最大のハイライトは Web ページの翻訳です。プラグインをインストールした後、英語の Web サイトを開くと、ワンクリックでバイリンガルになります。左側に原文、右側に訳文があり、特に読みやすいです。

英語のブログやMediumの記事、YouTubeの字幕を読むのによく使っています。バイリンガルなので英語の練習もでき、情報もすぐに得られるので一石二鳥です。

PDFの翻訳機能もとても便利です。 PDF を直接アップロードし、元の形式を維持したまま翻訳後にダウンロードします。これは学生パーティーにとって非常にフレンドリーです。

欠点は、翻訳できるのは Web ページのみであり、会話は翻訳できないことです。リアルタイムの会話翻訳が必要な場合は、他のツールを使用する必要があります。

## 我的推荐

### 按场景推荐

- **学術論文/専門文書翻訳**: DeepL
- **多言語翻訳/日常使用**: Google 翻訳
- **説明が必要なクリエイティブコピー/翻訳**: GPT-4
- **中英翻訳/国内コンテンツ**: Tencent Translation
- **外国語のウェブサイト/情報を読む**: Caiyun Xiaoyi

### 按预算推荐

- **予算 0**: Google 翻訳 + Caiyun Xiaoyi の組み合わせ
- **予算 10 ユーロ/月**: DeepL 有料版
- **コスト的には悪くない**: DeepL + GPT-4 の組み合わせ

### 按用户群体推荐

- **学生/研究者**: DeepL + Caiyun Xiaoyi
- **越境電子商取引**: DeepL
- **海外旅行**: Google 翻訳
- **コンテンツクリエーター**: GPT-4 + DeepL
- **国内ユーザー**: Tencent 翻訳 + Caiyun 翻訳

## 总结

AI 翻訳ツールは現在非常に人気がありますが、完璧なツールはなく、現在のニーズに最適な選択肢があるだけです。

- **品質が必要**: DeepL を選択してください
- **さらに多くの言語が必要**: Google 翻訳を選択します
- **理解が必要**: GPT-4 を選択してください
- **中国語を上手に話したい**: Tencent 翻訳者を選択してください
- **Web ページを読みたい**: Caiyun Xiaoyi を選択してください

私の提案は、お金と心配を節約するために日常の翻訳には Google 翻訳または Tencent 翻訳を使用し、専門的な翻訳には DeepL を使用し、複雑なタスクには GPT-4 を使用することです。最良の結果を得るには、いくつかのツールを組み合わせて使用​​します。
