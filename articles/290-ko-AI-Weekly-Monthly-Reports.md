## AI로 보고서를 쓰는 장점

### 1.速度快
이전에는 수동으로 주간 보고서를 쓰는데 30분~1시간 걸렸는데, 이제 AI면 10분 만에 가능.

### 2.構造清晰
AI가 자동으로 정보를 명확한 구조로 정리해줘서, 수동으로 서식 지정할 필요 없음.

### 3.전문적인 언어
AI가 전문적이고 격식 있는 언어로 작성해서, 표현 부족을 worry할 필요 없음.

### 4.수정 가능
AI가 쓴 것을实际情况에 맞게 수정할 수 있어서, 처음부터 쓰는 것보다 훨씬 쉬움.

## 구체적인 방법

### 1단계: 이번 주/월 업무 정리

보고서를 쓰기 전에 이번 주/월 한 일을 먼저 나열:

**정리 방법**:
- 완료한 프로젝트
-参与的 회의
- 만난 문제 (선택)
- 다음 주/월 계획

**팁**:매일 간단히 메모하는 것이 좋음, 그렇지 않으면 보고서 쓸 때 구체적인 일을 기억 못함.

### 2단계: 맞는 AI 도구 선택

| 도구 | 특징 | 적합한 |
|------|------|--------|
| Kimi | 긴 텍스트 처리 강함 | 상세하고 内容이 많은 보고서 |
| 통义千问 | 안정적 신뢰 |常规 업무 보고서 |
| 문心一言 | 웹 검색으로 데이터 보완 | 데이터가 필요한 보고서 |
|讯飞星火 | 음성 입력 | 음성派 좋아하는 사람 |

### 3단계: 명확한 지시

가장 중요한 단계! 지시가 좋을수록 AI 결과도 좋음.

**만능 공식**:
```
당신은 [직업]입니다. [주간/월간] 보고서를 쓰는 것을 도와주세요. 내용: [구체적인 내용], 요구사항: [형식/길이/톤]
```

**구체적인 예시**:

**시나리오1: 주간 보고서 쓰기**
```
당신은 제품 매니저입니다. 이번 주 업무 보고서를 써주세요.内容包括: 1.완료된 요구사항 검토 2.개발 협업 상황 3.사용자 피드백 처리 4.다음 주 계획. 요구사항: 명확한 구조, 각 항목 50자 이내, 격식 있는 톤.
```

**시나리오2: 월간 보고서 쓰기**
```
당신은 마케팅 전문가입니다. 월간 업무 총括를 써주세요.内容包括: 1.이번 달 완료한 주요 업무 2.성과(데이터 있으면 좋음) 3.현재 문제 4.다음 달 계획. 요구사항: 데이터 사용, 전문적인 격식 있는 언어, 약 800자.
```

### 4단계: 확인하고 수정

AI가 쓰면 반드시 다음 항목 확인:

1. **사실 확인**: AI 쓴 프로젝트가 실제로 자신의 것인지 확인
2. **데이터 확인**: 데이터가 있으면 정확성 확인
3. **톤 조정**:自己和 상사习惯에 맞춰 톤 조정
4. **개인정보 확인**: 민감한 정보 유출 주의

## 자주 묻는 질문

### Q1: AI 쓴 보고서가 너무笼统的怎么办?
A: 더 구체적인 업무 내용을 제공. 상세할수록 AI도 더 구체적으로 씀.

### Q2: 상사가 AI 쓴 것一眼就能看出吗?
A: 적절히 확인하고 수정하면基本上没人能看出. 핵심은 자신의 실제 업무 내용을 추가하는 것.

### Q3: AI로 가짜 업무 내용을 만들어도 되나요?
A: 절대 안 됨! 주간/월간 보고서는实事求是, AI는 정리하고 표현하는 것을 도와주는 것, 내용을 만들어내는 것이 不是.

### Q4: 회사마다 보고서 요구사항이 다르면怎么办?
A: 지시사항에 회사 요구사항을 명확히 알려주면 됨, 예: "회사는简洁한 보고서 요구, 각 항목 3줄 이내."
