## Astuce 2 : Comment écrire le mot d'invite

### Ne pas utiliser trop longtemps

DeepSeek n'est pas la même chose que ChatGPT.

ChatGPT écrit une longue liste de mots d'invite et les lit tous.

Si vous écrivez DeepSeek trop longtemps, il déraillera facilement.

**Écriture correcte** :
```
Écrivez un article sur la communication sur le lieu de travail, d'environ 1 000 mots. Le public cible est constitué de nouveaux arrivants qui viennent de rejoindre le poste. Cela doit être pratique et pas trop théorique.
```

**Mauvaise façon d'écrire** :
```
Vous êtes un expert senior en ressources humaines avec 10 ans d'expérience en recrutement et êtes doué pour gérer les problèmes de relations de travail. Veuillez rédiger un article selon les exigences suivantes : d'abord... deuxièmement... puis... enfin...
```

C'est trop long, DeepSeek a lu la fin et a oublié le début.

### Demandez étape par étape

Posez des questions complexes séparément.

Par exemple, si vous souhaitez rédiger un article :

Première étape :
```
Aidez-moi à rédiger un aperçu de l'article « Compétences en communication sur le lieu de travail », 5 parties
```

Après que DeepSeek ait donné les grandes lignes :

Deuxième étape :
```
Sur la base du premier plan, rédigez un paragraphe détaillé de 200 mots
```

Troisième étape :
```
Continuez à écrire la deuxième partie du plan
```

Cela fonctionne bien mieux que de poser toutes les questions en même temps.

### Effacer le format

DeepSeek peut parfois en faire trop et écrire un tas de choses dont vous n'avez pas besoin.

Dites-lui exactement quel format vous souhaitez.

```
Veuillez l'afficher sous forme de tableau. Chaque élément de comparaison doit inclure : la fonction, les performances de DeepSeek, les performances de ChatGPT
```

```
Veuillez expliquer en points, pas plus de 5 éléments, chaque élément ne doit pas dépasser 50 mots.
```

Ce ne sera pas gratuit de jouer.

## Astuce 3 : Utilisation basée sur des scénarios

### Rédiger une copie

DeepSeek est très doué pour rédiger des textes chinois.

Surtout le genre terre-à-terre.

Par exemple, rédiger une copie pour Moments :
```
Aidez-moi à écrire à un cercle d'amis en leur disant que mes collègues m'ont offert du thé au lait et des gâteaux pour le thé de l'après-midi cet après-midi. Soyez un peu drôle.
```

DeepSeek vous donnera une version intéressante.

### Écrire du code

Il peut également être utilisé lors de l’écriture de code.

Mais l'effet réellement mesuré n'est pas aussi bon que celui de Claude.

L'écriture du code de Claude est plus standardisée et ses commentaires sont plus clairs.

DeepSeek rencontre parfois des problèmes mineurs lors de l’écriture du code, il est donc recommandé de le vérifier deux fois.

###Écrire un article

DeepSeek peut rédiger des articles sur des comptes officiels, des réponses sur Zhihu et la rédaction de Xiaohongshu.

Surtout pour le contenu qui nécessite un « sens Internet », le sens du langage de DeepSeek est très bon.

### traduire

La traduction de DeepSeek est plus naturelle que celle de Google Translate.

Ce n’est pas une traduction automatique rigide, mais elle se lit comme un discours humain.

```
S'il vous plaît, aidez-moi à traduire ce paragraphe de l'anglais vers le chinois. Cela devrait être naturel et non littéral.
```

Cela fonctionne très bien.

### Réflexion

Vous ne trouvez pas de sujet à choisir ? Vous ne savez pas comment prendre une décision ?

Laissez DeepSeek vous aider à réfléchir.

```
Je suis un blogueur sur le lieu de travail. J'écris principalement sur la croissance du lieu de travail et les compétences en communication. Veuillez me donner 10 sujets parmi lesquels choisir.
```

Cela vous donnera un tas d'idées.

## Astuce 4 : Guide pour éviter les pièges

### Ne comptez pas trop

DeepSeek dit parfois de sérieuses bêtises.

Il fabrique des données et des devis pour paraître professionnels, mais il s’agit en réalité de faux.

**Solution** :
- Vérification manuelle des informations importantes
- Implique une vérification multi-angle des connaissances professionnelles
- Ne copiez pas directement sur le document officiel

### Problèmes sensibles

DeepSeek procède à une révision du contenu et il y a certaines questions auxquelles il ne répond pas.

Par exemple :
- Comment contourner le paiement
- Comment cracker un logiciel
- Événements historiques sensibles

Il rejettera ces questions ou vous donnera un discours officiel.

### Restrictions contextuelles

La fenêtre contextuelle de DeepSeek n'est pas aussi longue que celle de Claude.

À la fin de la rédaction d’un long article, vous risquez d’oublier ce qui a été dit plus tôt.

**Solution** :
- Écrivez en sections, chaque section doit contenir moins de 2 000 mots.
- Répétez les informations importantes au début de chaque paragraphe
- Résumez manuellement le contenu précédent et alimentez-le si nécessaire

### Ne posez pas de questions sur des choses trop nouvelles.

Les données d'entraînement de DeepSeek ont ​​une date d'expiration.

On ne sait peut-être pas ce qui se passera après 2024.

Demandez-lui « Quelle grande chose s'est produite récemment » et il vous donnera une réponse obsolète.

## Astuce 5 : Utilisation avancée

### Mot d'invite du système

DeepSeek prend en charge les mots d'invite du système.

Cliquez sur Paramètres dans le coin supérieur droit pour définir le comportement par défaut de l'IA.

Par exemple :
```
Vous êtes un consultant professionnel en entreprise. Répondez aux questions de manière concise et efficace et utilisez des données et des cas pour étayer vos opinions.
```

Une fois définies, toutes les conversations ultérieures prendront effet.

### Modèle de conversation

Des mots d’invite utiles peuvent être enregistrés sous forme de modèles.

DeepSeek ne dispose actuellement pas de fonctionnalité de modèle officielle, mais vous pouvez :

1. Écrivez-le dans un document
2. Copiez et collez si nécessaire
3. Ou prenez une capture d'écran et enregistrez-la

### Appels d'API

Les développeurs peuvent y accéder via l'API.

L'API de DeepSeek est moins chère qu'OpenAI.

La documentation est clairement écrite et conviviale pour les développeurs.

### Déploiement local

Open source signifie que vous pouvez télécharger le modèle et l'exécuter localement.

Exigences de configuration :
- Carte graphique avec au moins 16 Go de mémoire vidéo
- Mémoire 32 Go ou plus
- Espace disque dur 100 Go+

Les ordinateurs ordinaires ne peuvent pas l'exécuter, il est donc recommandé d'utiliser les services cloud.

##FAQ

**Q : DeepSeek facture-t-il ? **

R : La version Web est entièrement gratuite. Les appels API sont également bon marché, bien inférieurs à ceux d’OpenAI.

**Q : Lequel est le meilleur, DeepSeek ou ChatGPT ? **

R : Regardez la scène. DeepSeek est meilleur pour écrire du contenu chinois et Claude est meilleur pour écrire du code. Vous pouvez l'essayer pour un usage quotidien.

**Q : Les données DeepSeek sont-elles sécurisées ? **

R : Les données de la version Web seront téléchargées sur le serveur. Les données de la version de déploiement local sont entièrement locales et plus sécurisées.

**Q : DeepSeek remplacera-t-il les humains ? **

R : Non. L’IA est un outil et non un substitut. Ceux qui apprennent à utiliser l’IA élimineront ceux qui ne peuvent pas l’utiliser.

**Q : DeepSeek doit-il contourner le pare-feu ? **

R : Non, c’est directement connecté en Chine.

**Q : DeepSeek a-t-il une application ? **

R : Actuellement, il n’existe qu’une version Web et aucune application officielle.

**Q : DeepSeek peut-il générer des images ? **

R : Non. DeepSeek est un modèle de texte et ne peut pas générer d'images.
