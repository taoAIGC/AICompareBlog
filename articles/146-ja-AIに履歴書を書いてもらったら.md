# AIに履歴書を書いてもらったら、面接の機会が2倍になった

---

## 1

従弟の肖張さんは去年就職活動をしていました。30通以上の履歴書を送りましたが、一つも返事がありませんでした。

僕のところに来て言いました。「兄さん、履歴書に問題がないか見てくれ。」

開けてみた。おったまげた。
「会社関連の業務を担当」→どの会社？どんな業務？
「コミュニケーション能力が強い」→どの程度？例は？
「オフィスソフト使える」→WordかExcel？どれくらいのレベル？

言いました。「履歴書書いてるんか、謎かけか？人事さんは混乱するよ。」

言いました。「本当に書き方が分からない。大学でも特に何もしてない。」

言いました。「大丈夫、AIに助けてもらおう。」

## 2

ほとんどの人は二つの極端な方法で履歴書を書きます。

書くことが何もないと思案中、清水のようになりだけ。
書きすぎで濃すぎる、3秒で閉じたいくらい。

実は、**良い履歴書 = 経験 + 成果 + 数字**。

経験がない？大学の課題、サークル活動、アルバイトでもOK。
成果がない？「Xを担当」は「Xやって、XをX%改善した」より弱い。
数字がない？「30%改善」は「良い結果」より効果的。

どう書く？AIが助けてくれます。

## 3

### AIに履歴書を書いてもらう方法

**ステップ1：経験を全部出す**

フォーマットを気にせず、したことを全部書いて：

「学生会の広報部で、ポスター制作と公式アカウントの記事書いてました。」
「夏の間に電子工場で、ライン作業やってました。」
「近所の子供に数学教えてました。」
「淘宝で网店やって、ゲーム装备売ってました。」

**ステップ2：AIに履歴書言葉に書き直しさせる：**

```
あなたは10年経験がある採用担当です。履歴書を最適化してください。
私は新規卒業生です。以下は私の経験です：
[経験を貼り付け]

履歴書フォーマットで書き直し、要件：
1. 各項目は「やったこと」＋「結果」を含む
2. 動詞で始める
3. 定量化できる成果を重視
4. 1ページに収めて
```

**ステップ3：職種に合わせて微調整**

技術職に応募？プログラミング、プロジェクト、問題解決能力を強調。
運営職に応募？計画、データ分析、ユーザーコミュニケーションを強調。
営業職に応募？コミュニケーション能力、売上実績、人との関係を強調。

全部同じテンプレートより、ピッタリ量身定制の方がいい。

## 4

### AI面接準備の方法

履歴書書いた。面接が本番です。

很多人在面也不知道準備什么。要么干等着紧张，要么背一堆废话。

其实**面试准备 = 常见问题 + 你的故事 + 模拟练习**。

**AI帮你列常见问题：**

```
你是一个资深面试官，帮我预测这个岗位的面试问题：
岗位：新媒体运营
要求：负责公众号、微博内容策划，粉丝增长

请列出10个最可能被问到的问题，并给出回答思路。
```

拿到问题后挨个准备，把你的故事套进去。

**你的故事怎么讲：**

STAR法则了解一下：
- **S**ituation（情境）：当时什么情况？
- **T**ask（任务）：你的目标是什么？
- **A**ction（行动）：你具体做了什么？
- **R**esult（结果）：结果怎么样？

比如被问到"你觉得你沟通能力怎么样"：

"（S）大学时我发现社团活动报名人数很少。（T）我想提高到100人。（A）我做了三件事：1）设计更吸引人的海报；2）逐个联系社团负责人；3）在校园群发通知。（R）最终报名人数达到156人，比之前多了60%。"

有情境有行动有数据，比干巴巴说"我沟通能力强"有力多了。

## 5

### 面试紧张怎么办

我表弟面试前紧张得睡不着。

我说："你紧张啥？大不了就是不过呗。"

他说："就是害怕嘛。"

我说："教你个办法，面试前把AI当成面试官，先练三遍。"

**模拟面试提示词：**

```
你是一个面试官，面试岗位是[岗位名]。
请向我提出5个常见面试问题，每次只问一个，等我回答后你再点评，然后问下一个。
态度可以严格一点，指出我回答的问题。
```

跟AI练一遍，心里就有底了。真正面试的时候，发现"这些题我练过"，紧张感少一半。

## 6

### 谈薪资怎么让AI教

薪资是面试最难开口的部分。

低了亏自己，高了怕黄掉。

**让AI帮你做薪资调研：**

```
我要面试[城市]的[岗位]，请问这个岗位的平均薪资是多少？
请给出不同经验级别的薪资范围。
```

心里有个数之后，怎么开价也有底气。

**让AI帮你模拟谈薪资对话：**

```
我面试的公司给我开了8000一个月，我想要10000。
请模拟一段谈薪资的对话，教我怎么开口，怎么说服对方。
```

提前练练，真的到了那个场合不至于说不出话。

## 7

### 面试后怎么跟进

面完试，很多人就是干等着。

其实**面试后跟进 = 感谢消息 + 复盘总结**。

**感谢消息怎么写：**

```
我今天参加了[公司]的[岗位]面试，面试官是[名字]。
请帮我写一段100字以内的感谢消息，表达我对这个岗位的热情，以及期待回复。
```

**复盘怎么写：**

每次面试完，花10分钟记下来：
- 哪些问题回答得好？
- 哪些问题回答得烂？
- 下次怎么改进？

我表弟按这个方法练了一个月，后面几场面试一个比一个顺。

### 最后说几点

**第一，简历是敲门砖，不是你的全部。**

写得好不代表能力强，但写得烂可能连面试机会都没有。花点时间打磨不亏。

**第二，AI是辅助，不是替代。**

AI帮你整理、帮你优化、帮你练习，但最终上场的是你自己。该记的要记，该练的要练。

**第三，面不上很正常，别灰心。**

找工作是个概率事件，面10家成1家就赚了。每次面试都是学习机会，复盘多了，成功的概率就上去了。
