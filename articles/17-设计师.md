## 设计师的AI组合

### 场景一：设计灵感收集

**输入提示词**：
> "帮我找一些优秀的APP登录界面设计案例，要求简洁、现代、有科技感"

**Perplexity版**：
```
优秀登录界面设计参考：
1. Google登录页：极简设计，Google Logo+输入框
2. Apple登录：生物识别+Apple ID
3. Spotify：暗色背景，品牌色强调
4. Figma：留白+插画
[来源：Dribbble, Behance, Awwwards]
```

**Claude版**：
```
登录界面设计趋势：
1. 极简主义：去除多余元素，只保留核心
2. 渐进式表单：一步步引导用户填写
3. 暗色模式：保护隐私，视觉冲击
4. 生物识别：指纹、面容识别代替密码
5. 社交登录：降低注册门槛
```

### 场景二：设计文案撰写

**输入提示词**：
> "帮我写一段APP启动页的设计说明，要说明设计理念、配色选择、交互设计。"

**ChatGPT版**：
```
设计说明：

1. 设计理念
   - 主题：「连接美好」
   - 风格：简约、温暖、年轻

2. 配色选择
   - 主色：品牌蓝（#1890FF）
   - 辅助色：浅蓝、白色
   - 渐变效果：增加层次感

3. 交互设计
   - 滑动切换：支持左右滑动
   - 自动跳过：3秒后自动进入首页
   - 动效：渐显+轻微弹跳

4. 适配说明
   - iOS和Android统一设计
   - 适配刘海屏和全面屏
```

**Kimi版**：
```
姐妹们！这款APP的启动页设计真的绝了😍

设计亮点：
1️⃣ 渐变蓝背景，看着就很舒服
2️⃣ 动态水波纹效果，炒鸡有质感
3️⃣ 文案简短有力，一眼就看懂

配色是那种清新的蓝白色系，
给人年轻、有活力的感觉！

动效也做得很细腻，
滑动的时候有轻微的回弹，
手感特别好～
```

### 场景三：设计评审准备

**输入提示词**：
> "帮我准备设计评审可能被问到的问题，以及如何回答"

**Claude版**：
```
设计评审常见问题及回答：

1. 为什么选择这个颜色？
   → 参考品牌色卡，说明色彩心理学依据

2. 这个间距为什么是8px？

3. 适配不同屏幕尺寸怎么办？

4. 加载状态怎么处理？

5. 为什么不用更常见的交互方式？
   → 对比分析，说明我们的方案更优
```
