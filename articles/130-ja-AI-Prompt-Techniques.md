## 同じAIなのに、人によって結果が天と地

私有一个朋友跟我同时开始用ChatGPT。

三个月后，他说："AI也就那样吧，没什么用。"

我：？？？

我看了他的对话记录，差点没背过气去。

他问ChatGPT的问题是这样的：
> "帮我写篇文章"

我的问题是这样的：
> "我是一名前端开发者，需要写一篇面向初学者的React入门教程，目标是完全没有前端经验的人，字数在2000字左右，语言风格要轻松幽默，多用比喻，请给出完整的markdown格式文章。"

结果就是——他能写出"流水账"，我能写出"爆款文"。

问题出在哪？**提示词（Prompt）。**

## 什么是Prompt？用大白话来说就是"你怎么问"

Prompt就是你跟AI说的话。

听起来很简单，但90%的人都不会"问"。

最典型的几个坑：

### 坑1：问得太笼统

> "帮我写个文案"

AI：？？？什么产品？给谁看？什么调性？要长要短？

### 坑2：缺少背景信息

> "帮我分析一下这个市场"

AI：？？？哪个市场？什么数据？分析给谁看？

### 坑3：期望不明确

> "给我点建议"

AI：？？？什么建议？职业？感情？投资？

这些问题AI不是不能回答，是**你不说清楚，它就猜，猜错了就全是废话**。

## 写好提示词的5个核心要素

我用了一个月时间，把自己的提示词习惯全改了。

现在跟AI对话，基本上一遍就能得到80%满意的结果。

### 1. 身份：你是谁

告诉AI"你是谁"，它才能用对的语气和角度跟你说话。

**❌ 错误示范**：
> "分析一下这个项目的风险"

**✅ 正确示范**：
> "我是一名正在考虑创业的产品经理，你需要从投资人的角度分析这个项目的风险"

### 2. 背景：要解决什么问题

给AI上下文，它才能理解你的处境。

**❌ 错误示范**：
> "帮我写个简历"

**✅ 正确示范**：
> "我有5年电商运营经验，最近在找工作，想应聘用户增长岗位，需要一份突出数据分析能力的简历"

### 3. 任务：具体要做什么

越具体，AI越不容易跑偏。

**❌ 错误示范**：
> "给我点建议"

**✅ 正确示范**：
> "请给我5个提高用户留存率的具体策略，每个策略需要有执行步骤和预期效果"

### 4. 格式：输出要什么样

告诉AI输出的形式，省去你再加工的时间。

**❌ 错误示范**：
> "介绍一下Python"

**✅ 正确示范**：
> "用表格形式对比Python和JavaScript的5个核心区别，包括语法、适用场景、性能、学习难度、生态"

### 5. 限制：不能做什么

加一些约束，AI的回答会更精准。

**❌ 错误示范**：
> "写一段产品描述"

**✅ 正确示范**：
> "写一段智能手环的产品描述，不超过100字，强调睡眠监测功能，避免使用'革命性'、'领先'这类夸张词汇"

## 我常用的几个"万能提示词模板"

### 模板1：学习新东西

```
我想学习[领域/技能]，我是完全的初学者，没有任何基础。
请用最简单易懂的方式解释核心概念，
给出3个实际的应用场景，
推荐1个入门级资源。
语言风格要生动有趣，不要太学术。
```

### 模板2：写东西

```
我需要写一篇关于[主题]的[类型：文章/邮件/文案/报告]。
目标读者是[具体人群]，
核心观点/目的是[xxx]，
字数控制在[xxx]字，
风格要求[xxx：专业/轻松/幽默/正式]。
请先给我3个不同角度的大纲，我选好再展开。
```

### 模板3：解决问题

```
我遇到了[问题描述]。
我已经尝试过[xxx方法]，但效果是[xxx]。
请帮我分析可能的原因，
给出3个不同的解决方案，
并说明每个方案的优缺点和实施难度。
```

### 模板4：获取建议

```
我正在考虑[决定/选择]，选项有A、B、C。
我的情况是[背景信息]，
我更看重[xxx因素]，
请帮我分析每个选项的利弊，
给出你的建议和理由。
```

## 不同AI，提示词要"因材施教"

我用过的AI少说也有十几个，发现一个规律：**不同的AI，脾气不一样。**

### ChatGPT

- 喜欢结构化、清晰的指令
- 对英文提示词反应更好
- 创意类任务表现优秀

### Claude

- 长文本处理能力强
- 喜欢"角色扮演"类提示词
- 分析类任务表现突出

### DeepSeek

- 中文理解能力强
- 不需要太复杂的提示词
- 代码和逻辑类任务很能打

### Kimi

- 长对话能力好
- 多轮对话中会"记住"前面的内容
- 文件分析能力强

## 进阶技巧：让AI"思考"再回答

有些问题，直接问效果不好。

这时候可以加一句：
> "请先思考一下，再给我答案"

或者更高级的：
> "请分三步思考这个问题：第一步分析问题本质，第二步列出可能的方案，第三步给出最终建议"

这招特别适合：
- 需要逻辑推理的问题
- 需要创意发散的问题
- 需要权衡利弊的决策

## 我的血泪教训

### 教训1：别舍不得删对话

AI有"记忆疲劳"。对话太长，它会"忘"前面的内容。

我的做法是：每个新任务，开新对话。

### 教训2：先让它"确认理解"

复杂任务，我会先问：
> "请复述一下我的需求，确保你理解了"

它确认没问题了，再让它执行。

### 教训3：多轮迭代是正常的

别期望AI一次就给你完美答案。

我的标准流程是：
1. 第一遍：要框架/大纲
2. 第二遍：填充内容
3. 第三遍：修改细节

通常3轮之内能得到满意的结果。

## 总结：AI是工具，你才是驾驶员

用AI快一年了，我最大的体会是——

**AI再强，也只是个工具。工具好不好用，取决于使用它的人。**

同样的ChatGPT，有人能靠它月入十万，有人觉得"就那样"。

区别不在AI，在于**你会不会问**。

学会了写提示词，你会发现：
- AI不只是能帮你"偷懒"，而是能帮你"思考"
- 不是AI太笨，是你没说明白
- 好的提示词 = 好的沟通 = 高效的结果

从今天开始，每次跟AI对话之前，先问自己一句：
"我把话说清楚了吗？"
