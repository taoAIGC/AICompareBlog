---

## 시나리오1: 글쓰기

**사용 방법**:
DeepSeek에게 니eds를 말해줘, 예:
"Python 학습 노트一篇 써줘, 1000자 정도, 초보자向け"

**효과**:
DeepSeek는 数초에 结构清晰、内容准确的 글 쓸 수 있어, 니意愿에 따라 수정 가능.

**팁**:
- 글 쓸 때 用途말해줘 (블로그/과제/업무보고)
- 글자 수 요구
- 스타일 지정 (전문적/캐주얼/유머)

---

## 시나리오2: 코드 지원

**사용 방법**:
"Python 크롤러 하나 써줘, 웹사이트 이미지 다운로드하는 거"

또는:
"이 코드 에러 있어?帮我看一下"

```python
# 问题 코드를 DeepSeek에게
def get_data():
    return a + b  # a와 b가 정의 안 됨
```

**효과**:
DeepSeek는 코드 쓸 수 있을 뿐더러, バグ 찾아주고 코드 로직 설명해줘.

**팁**:
- 프로그래밍 언어와 버전 명시
- 구체적인 요구사항 설명
- 에러 메시지 첨부

---

## 시나리오3: 학습 질문

**사용 방법**:
"머신러닝이 뭐야, 간단한 예로 설명해줘"

또는:
"이 수학 문제 어떻게 풀어?"

**결과**:
DeepSeek는 질문에 참을성 있게 답변하고, 간단한 예로 복잡한 개념을 이해하도록 도와줘.

**팁**:
- 모르면 계속 물어: "더 자세히 알려줘"
- "예를 들어줘"
- "이거랑 저거랑 뭐가 달라?"

---

## 시나리오4: 텍스트 편집

**사용 방법**:
쓴 텍스트를 보내줘:
"이거 더 전문적으로 만들어줘"

원문: 이거 좋네, 팔고 싶어
수정: 본 제품은 우수한 성능과 광활한 시장 전망을 가지고 있습니다

**효과**:
DeepSeek는 평범한 文을 전문적인 카피로 바꿔줄 수도, 너무Formal한 文을 더 접근하기 쉽게 만들어줄 수도 있어.

**팁**:
- 대상 독자 명시
- 원하는 스타일 지정
- 편집인지 새로 쓰기인지 명확히

---

## 시나리오5: 번역

**사용 방법**:
"이거 중국어로 번역하고, 어려운 단어 설명해줘"

또는:
"영어로 비즈니스 이메일 써줘"

**결과**:
DeepSeek의 번역 품질은 높고, 단어도 가르쳐줄 수 있어.

**팁**:
- "직역" 또는 "의역" 요구 가능
- 중요 단어.hljs;ighlight시켜달라고 할 수 있어
- 문법 설명 요구 가능

## 핵심 요령

DeepSeek 잘 쓰는 방법은 3가지:

1. **명확히** - 뭐想要的지 명확히
2. **계속 물어** - 마음에 안 들면 수정 요청
3. **프롬프트 배워** - 좋은 프롬프트 = 좋은 결과
