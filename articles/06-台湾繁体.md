## 每天開不完的標籤頁

我是阿偉，在台北一家新創公司擔任產品經理。

每天工作打開瀏覽器，我的標籤頁都是這樣的：

```
ChatGPT │ Claude │ Kimi │ Perplexity │ 豆包 │ Gmail │ Slack │ Figma │ ...
```

寫PRD要找ChatGPT，畫原型要找Claude做文案優化，查竞品要開Perplexity，做用户访谈要找Kimi處理中文...

**我每天在6個AI工具之間來回切換。**

有一次，我發現自己對著同一個問題，問了ChatGPT三次。

因為標籤頁太多，我完全不記得哪個是新的，哪個是已經問過的。

那一刻我決定：**我需要改變。**

## 一個插件，改變了我的工作流程

同事推薦了「AI比一比」。

老實說一開始我覺得，不就是多了個聚合頁面嗎？

**結果我錯了。**

改變從第一天就開始了。

## 使用前 vs 使用後

**使用前（典型的一天）**：
```
9:00-9:30  在ChatGPT上問「怎麼寫用户故事」
9:30-10:00 切換到Claude問同樣的問題
10:00-10:30 打開Kimi優化中文表達
10:30-11:00 手動對比三個回答，整理成自己的文檔
→ 2小時過去了，只完成了一個需求文檔的開頭
```

**使用後**：
```
9:00-9:15 打開「AI比一比」，同時向ChatGPT、Claude、Kimi提問
9:15-9:30 三欄並排，對比三個AI的回答，取其精華
9:30-10:00 一份高質量的需求文檔完成了
→ 同樣的工作量，從2小時壓縮到1小時
```

**節省了50%的時間。**

## 為什麼PM需要多AI對比？

### 場景一：寫PRD

假設你要寫一個「用户收藏功能」的PRD。

同樣的問題，不同AI的視角完全不同：

**ChatGPT**（結構化思維）：
```
## 用户故事
- 作為用户，我希望收藏喜歡的商品，以便以後快速找到
## 功能描述
- 收藏入口：商品詳情頁、購物車、搜索結果頁
## 驗收標準
- 點擊收藏後，圖標變為已收藏狀態
```

**Claude**（用户視角）：
```
## 用户痛點
- 用户遇到喜歡的商品，但當時不打算購買，事後找不到
## 解決方案
- 提供一鍵收藏功能，降低用户決策成本
## 交互設計
- 收藏後有即時反饋，如 toast 提示或圖標變化
```

**Kimi**（中文優化）：
```
## 需求背景
姐妹們！有沒有遇到過這種情況：看到喜歡的衣服，當時沒買，
回頭怎麼也找不到鏈接了？我們的收藏功能就是來解決這個痛點的！
## 核心功能
- 一鍵收藏：點一下就行，不用跳轉
- 快速找到：我的收藏裡一目了然
- 降價提醒：收藏的商品降價了，第一時間通知你
```

三個AI的回答各有側重：
- ChatGPT 給了標準結構
- Claude 補充了用户痛點分析
- Kimi 讓表達更接地氣

**融合起來，就是一份完美的PRD。**

### 場景二：竞品分析

分析竞品App的一個功能設計：

**Perplexity**（搜索最新信息）：
> "根據用戶評價，竞品A的收藏功能上線3個月，
> 收藏轉化率提升了15%。用戶普遍反饋'找商品更方便了'。"

**ChatGPT**（總結分析）：
> "竞品A的收藏功能特點：1. 一鍵收藏 2. 分類管理 
> 3. 降價提醒 4. 社交分享。優點是交互流暢，
> 缺點是收藏入口較深。"

**Claude**（建議參考）：
> "從產品設計角度，建議：1. 收藏入口放在更顯眼的位置 
> 2. 增加收藏夾分類功能 3. 提供批量操作。"

### 場景三：用户訪談提綱

**ChatGPT**（標準模板）：
```
1. 您平時有收藏商品的習慣嗎？
2. 您通常在什麼情況下會使用收藏功能？
3. 您對現有竞品的收藏功能滿意嗎？
4. 您希望收藏功能增加什麼新特性？
```

**Claude**（深度問題）：
```
1. 能否描述一次您想收藏但最終沒有收藏的經歷？
2. 是什麼阻止您使用收藏功能？
3. 如果收藏的商品降價了，您希望以什麼方式收到通知？
4. 您會定期查看自己的收藏夾嗎？
```

**Kimi**（口語化）：
```
1. 姐妹們！大家平時買東西會收藏嗎？為啥呀？
2. 有沒有明明收藏了，結果再也找不到的情況？
3. 你們希望收藏夾能增加啥功能？
4. 收到降價提醒會買嗎？啥時候買？
```

## 核心功能詳解

### 1. 多站對比模式

一個頁面同時展示多個AI的回答。

### 2. 快捷鍵喚出

按 `Ctrl+M`（Windows）或 `Cmd+M`（Mac），立即彈出輸入框。

### 3. 劃詞查詢

瀏覽竞品網頁時，看不懂某個功能設計？

**選中文字** → 旁邊出現AI圖標 → 點擊 → 多個AI幫你分析。

### 4. 標籤頁模式

開啟「標籤頁模式」後，每次新建標籤頁都會自動打開多AI對比視圖。
