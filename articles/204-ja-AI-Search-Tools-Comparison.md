## AI比一比：マルチAI検索サポート

実は、もっと効率的な検索方法があります：**AI比一比で検索をサポート**。

### 传统的方法
```
1. Metaxを開く → 質問を検索
2. Perplexityを開く → 質問を検索
3. 天工を開く → 質問を検索
4. 渡り歩く → 面倒
```

### AI比一比的方法
```
1. Ctrl+MでAI比一比を開く
2. Metax + Perplexity + 天工AIを選択
3. 「2024年おすすめスマホ」と入力
4. 3つのAIが同時に答えを返す
5. 一番いいものを雰囲
→ 10秒で終了
```

 이렇게 하면:
- Metaxに中国語最佳の回答を生成してもらえる
- Perplexityに英語調査の結果を生成してもらえる
- 天工に綜合バージョンを生成してもらえる

1つのツールで完結、渡り歩く必要なし。
