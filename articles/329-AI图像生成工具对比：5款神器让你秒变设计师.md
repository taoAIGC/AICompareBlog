# 高级技巧

## 技巧1：组合使用

不同场景用不同工具：

- 创意阶段用Midjourney找灵感
- 确定方案后用DALL-E 3细化
- 批量生成用Stable Diffusion

## 技巧2：提示词优化

好的提示词是成功的一半：

```
主体描述 + 风格关键词 + 光线 + 色彩 + 构图
```

比如：
```
一只穿西装的猫，在办公室里工作，电影光效，暖色调，三分构图
```

## 技巧3：利用社区资源

Midjourney和Leonardo都有活跃的社区，多参考别人的提示词和参数，能快速提升效果。
