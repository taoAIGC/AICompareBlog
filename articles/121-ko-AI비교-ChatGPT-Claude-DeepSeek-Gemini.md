##はじめに（한국어）

저는 이 질문을 무수히 많이 받아왔습니다. "도대체 어떤 AI가 가장 좋죠?"

솔직히 이 질문을 볼 때마다 웃음이 나옵니다. 가장 좋은 AI란 존재하지 않습니다. 각 작업에 가장 적합한 AI가 있을 뿐입니다.

하지만 여러분들은 이런 둘러대는 이야기를 듣고 싶어하지 않는다는 것을 압니다. 그래서 저는まる 일주일을 들여 네 개의 주요 AI에게 같은 질문을 "가혹하게" 던지고 그들의 약점을 완전히 드러냈습니다.

이 기사는 허풍이 아니라 전부 실측 데이터입니다.

## 테스트 프레임워크

### 테스트 차원

각 AI를 평가하기 위해 5가지 핵심 차원을 설정했습니다:

| 차원 | 설명 |
|------|------|
| 논리적 추론 | 수학 문제, 논리 퍼즐, 인과 분석 |
| 크리에이티브 라이팅 | 카피라이팅, 스토리, 마케팅 플랜 |
| 코딩 능력 | Python, JavaScript, SQL |
| 중국어 이해 | 성어, 문화, 맥락 |
| 응답 속도 | 생성 시간(초) |

### 테스트 방법

같은 질문을 4개의 AI에 동시에 보내고, 답변을 기록하여 비교했습니다.
