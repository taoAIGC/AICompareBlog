## 具体的な使い方

### ステップ1：PPTテーマと内容を決定

AIを使う前に、基本的な内容を明確にしておく：

**事前に決めておくこと**：
- このPPTで何を伝えるか（テーマ）
- 誰に向けてか（対象者）
- どのような目的か（説得？紹介？報告？）
- 主要な内容は何か

**コツ**：紙やドキュメントに簡単なアウトラインを 作戦でOK。詳細でなくても、構造があればOK。

### ステップ2：適切なツールを選択

シーンに合わせて選ぼう：

- **ビジネス提案、上司への報告**：Gamma推奨、テンプレートが専門的
- **クリエイティブ展示、個人ブランド**：Tome推奨、スタイルが独特
- **日常業務、简单な報告**：美图AI PPTまたはCanva AI推奨
- **既存のOfficeドキュメントと連携必要**：iSlide AI推奨

### ステップ3：明確な指示を出す

これが最も重要！指示が正しいないと、AIは何を求めているかわからない。

**万能 формула**：
```
[テーマ]のPPTを作成してください。対象者は[誰]で、[どんなシーン]に使います。スタイルは[フォーマル/アクティブ/ 전문적]で、以下の内容を含めます：[主要章]
```

**具体例**：

**シーン1：仕事報告**
```
四半期の工作总结PPTを作成してください。対象者は部署の指導者と同僚で，月次例会報告に使います。スタイルはプロフェッショナルでシンプル、内容には以下を含めます：1.今四半期の業務完了状況 2.直面した挑战と解決策 3.来四半期の業務計画。データで語ることをお願いします。
```

**シーン2：プロジェクト提案**
```
新商品上市推广方案PPTを作成してください。対象者は市場部和運営チームで、プロジェクト立上げ会議に使います。スタイルはクリエイティブでアクティブ、内容には以下を含めます：1.市場背景 2.ターゲットユーザー像 3.推广策略 4.予算とスケジュール安排 5.预期効果。
```

**シーン3：培训课件**
```
新入社員入社培训PPTを作成してください。対象者は会社の新入社員で，入社培训会用です。スタイルは 친절で親しみやすい、内容には以下を含めます：1.会社紹介 2.企業文化 3.社内規則 4.部門分担 5.よくある質問FAQ。
```

### ステップ4：確認と调整

AIが生成したら、必ずこれらの点をチェック：

1. **内容の正確性**：AI書いた内容が実際の状況に合っているか確認
2. **論理の一貫性**：最初から最後まで読み通し、論理が连贯しているか
3. **画像のマッチ度**：挿し絵が適切か、替えられるものは替え
4. **ブランド一貫性**：会社VIがあれば、色やフォントがマッチしているか確認
5. **タイポ**：AIも文字を間違うことがあるので、最終確認を忘れずに

## よく使うPPT構成テンプレート

### 仕事報告PPT
```
1. 表紙（タイトル+報告者+日付）
2. 今期の業務概要（一言でまとめる）
3. コア成果（1〜3つの亮点、データ 있으면更好）
4. 詳細業務內容（箇条書きで説明）
5. 問題と解決策（問題解決能力を示す）
6. 次期計画（今後の業務重点）
7. 終了ページ（謝辞+質疑応答）
```

### プロジェクト提案PPT
```
1. 表紙（プロジェクト名+チーム）
2. 背景と課題（なぜやるか）
3. 解決策（あなたのプランは何か）
4. 市場分析（どのくらいの機会か）
5. 実行計画（どのようにやるか）
6. リソース必要量（どんなサポートが必要か）
7. 期待効果（どうなるか）
8. 次のアクション（即行動）
```

## よくある質問

### Q1：AIが生成したPPTが画一的で味けない怎么办？
A：生成後に微調整できる。例えば画像を変える、色を換える、レイアウトを調整するなどを通じて、自分色仕上げられる。また、複数のツールを試して自分に一番合うものを見つけるのも不错。

### Q2：AIが書いた文章が堅すぎる/柔らかすぎる怎么办？
A：指示に语气要求を明確に書ける。例えば「易懂な言葉で」や「正式なビジネスtoneで」など。生成後、自分でも修正できる。

### Q3：画像不满意可以换吗？
A：モチロンOK！AIが生成した画像はあくまで 参考なので、好みの画像に替换可能。大抵のツールが画像替换をサポートしている。

### Q4：プライバシーセキュリティが心配怎么办？
A：プライバシーポリシーを持つエンタープライズ向けツールを選べる。また、PPTに過度に 민감な情報を入れないようにすればOK。一般的な仕事報告レベルなら問題ない。
