## 什么是AI比一比？

**AI比一比** 是一个免费的 Chrome 浏览器插件。

它的核心功能很简单：**一次输入，多个AI同时回答**。

想象一下：你在一个输入框里打问题，按下回车，ChatGPT、Claude、Kimi、Perplexity、豆包同时给出答案。你可以在同一页面里对比它们的回答，选最好的。

这就是 AI比一比。

## 为什么新手需要它？

如果你刚开始用 AI 工具，你可能会困惑：

- ChatGPT 和 Claude 哪个好？
- Kimi 和豆包有什么区别？
- Perplexity 搜索和百度有什么不同？

**AI比一比让你不用纠结**——一次提问，同时看到所有AI的回答，高下立判。

## 第一步：安装插件（2分钟）

### 打开 Chrome 网上应用店

在浏览器地址栏输入：

```
https://chrome.google.com/webstore/detail/multi-ai/
```

或者直接搜索「AI比一比」或「Multi-AI」。

### 点击「添加至 Chrome」

页面会弹出确认对话框，点击「添加扩展程序」。

### 确认安装成功

安装完成后：
- 浏览器右上角工具栏会出现一个图标
- 如果没看到，点击拼图图标，找到「AI比一比」，点击图钉固定

**安装完成！** 整个过程大约2分钟。

## 进阶技巧（新手必看）

### 技巧1：快捷键唤醒

每次点击图标太麻烦？

**按 `Ctrl+M`（Windows）或 `Cmd+M`（Mac）**，随时唤出输入框。

### 技巧2：划词查询

在网上看到一段文字，想让AI帮你解释？

**选中文字** → 文字旁边出现AI图标 → 点击图标 → 多个AI同时帮你分析。

### 技巧3：配置常用AI

1. 点击右上角「设置」
2. 选择「默认AI组合」
3. 勾选你常用的AI（建议：ChatGPT + Claude + Kimi）
4. 保存设置

### 技巧4：标签页模式

如果经常需要对比AI回答，开启「标签页模式」：

1. 打开插件设置
2. 找到「标签页模式」
3. 开启

效果：每次新建标签页，都会自动打开 AI比一比面板。

## 新手常见问题

### Q1：AI比一比要钱吗？

**完全免费**。无需注册、无需付费、无使用次数限制。

### Q2：需要登录AI账号吗？

**不需要**。AI比一比是帮你同时访问多个AI工具，你仍然用各自的免费额度。

### Q3：响应速度慢怎么办？

AI比一比的响应速度取决于：
- 你选择的AI数量（选得越多越慢）
- 当前网络状况
- 各AI服务器的繁忙程度

建议：第一次用选2-3个AI，熟悉后再增加。

### Q4：手机能用吗？

目前仅支持 **Chrome 浏览器（桌面端）**。手机浏览器暂不支持。
