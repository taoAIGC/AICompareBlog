## 먼저 자신에게 물어보자

AI를 뭐에 쓸 거야?

이게 제일 중요해.

AI 도구마다 잘하는 게 다르거든.

코드写得好的사람, 글쓰는 사람, 그냥 놀면서聊天하는 사람, 엑셀 도와달라는 사람.

틀린 거 고르면 돈은 쓰고 효과는 없음.

## 먼저 내 니즈를 명확히 하자

### 시나리오1: 일상 대화, 질문, 글쓰기

그냥 놀면서聊聊天, 질문 하나, 글 하나 쓰는 거.

**추천: DeepSeek, ChatGPT**

이 두 개가 제일 쉬워. 브라우저 열면 바로 써.

DeepSeek는 공짜라서 안 쓸 이유 없잖아.

### 시나리오2: 코드 쓰기

프로그래머거나 프로그래밍 배울 거면.

**추천: Claude**

Claude가 제일 깔끔한 코드 써. 주석도 명확하고 논리도 명확.

ChatGPT도 되는데 가끔 게을러.

DeepSeek는 코드에 문제 있어, Claude가 낫다.

### 시나리오3: 중국어 콘텐츠 쓰기

웨이보 글, Zhihu 답변, 샤오홍쉬 캡션 쓰려면.

**추천: DeepSeek**

DeepSeek 중국어가 너무 좋아. 원어민이 쓴 거 같아.

ChatGPT 중국어는 가끔 번역투해서 읽기 불편함.

### 시나리오4: 긴 문서 처리

몇십 페이지 PDF 읽거나 긴 채팅 로그 분석하면.

**추천: Claude, Kimi**

이 두 개가 긴文本 처리 강함.

Claude는 100K 토큰까지 읽고, Kimi는 특히 긴文本으로 유명.

### 시나리오5: 번역

문서 번역하거나 외국인하고聊天하면.

**추천: DeepSeek**

DeepSeek 번역이 Google 번역보다 훨씬 자연스러움.

기계번역처럼 딱딱하지 않아, 사람이 쓴 거 같음.

### 시나리오6: 표 만들기, 데이터 처리

데이터 정리, 표 만들기, 데이터 분석하면.

**추천: ChatGPT, Claude**

이 두 개가 구조화된 데이터 처리 잘함.

수식 쓰기, 그래프 그리기도 DeepSeek보다 나음.

## 주요 AI 도구横向比较

### 무료 사용

| 도구 | 무료 여부 |
|------|----------|
| DeepSeek | 완전 무료, 웹 무제한 |
| ChatGPT | 무료版 제한 있음, Plus $20/월 |
| Claude | 거의 무료版 없음, Pro $20/월 |
| 통의천문 | 무료, 제한 있음 |
| 문심언어 | 무료, 제한 있음 |
| Kimi | 무료, 제한 있음 |

### 중국어 능력

DeepSeek > Claude > ChatGPT > 다른 중국 도구들

DeepSeek가 제일 자연스러운 중국어.

### 코드 능력

Claude > ChatGPT > DeepSeek

Claude가 제일 표준적인 코드 씀.

### 응답 속도

ChatGPT > Claude > DeepSeek

DeepSeek가 피크 시간대에 느림.

### 컨텍스트 길이

Claude > Kimi > ChatGPT > DeepSeek

Claude가 제일 긴 대화 기억함.

## 내 추천

### 학생들

먼저 DeepSeek 써, 공짜인데 안 쓸 이유 없어?

코드 쓸 일 생기면 Claude 써봐.

### 오피스 워커

매일 DeepSeek로 해, 글쓰기와 번역엔 충분.

중요한 문서는 Claude로, 품질更好.

### 프로그래머

바로 Claude 써, 고민 말고.

코드 쓴다면 Claude가 지금 제일 잘함.

### 컨텐츠 크리에이터

DeepSeek로 중국어 쓰고, Claude가 수정.

같이 쓰면 효과가 배가.

### 기업用户

API 쓰는 거 추천, 비용 통제 가능.

DeepSeek API가 OpenAI보다 쌈.

## AI比一比插件은 뭐가 돼?

AI 도구 너무 많은데, 결국 뭐가 좋음?

설명만으로는 부족하고, 실제로 써봐야 알 수 있음.

**AI比一比**插件이 바로 이거야.

설치하고 질문하면, Claude, GPT, DeepSeek가 동시에 답함.

탭 왔다갔다할 필요 없음.

一眼에哪个 답이更好 알 수 있어.

내가 글 쓸 때 쓰고 있음.

같은 질문, 4개 AI, 4개 답.

좋은 거 남기고, 안 좋은 거 삭제.

효율이 훨씬 높아.

Chrome插件이야, "AI比一比"로 검색하면 나옴.

완전 무료, 등록 필요 없음.

쓰는지 안 쓰기는 네 자유, 난 실제로 씀.

## 돈 낭비하지 마

很多人一开始就买ChatGPT Plus.

그러고 보니까 무료版도 충분함.

손해 아니야?

먼저 무료版 써보고, 니즈를 명확히 해.

그 다음에 돈 낼 필요가 있는지 판단해.

지금은 AI 도구 경쟁이 치열해서, 각 사가 用户争夺战.

무료版도大概够用,花钱的不一定更好.

## 결론

AI 도구에 絶対的好坏는 없어.

적합한지 안 적합한지뿐.

매일 코드 쓰는 사람? Claude가 best.

매일 중국어 쓰는 사람? DeepSeek가 best.

흐름에 휩쓸리지 마.

먼저 내가 뭐가 필요한지 생각해.

그 다음에 맞는 도구를 찾아.

插件은 보조 수단, 결정권은 네手里에 있어.
