## Tipp 2: So schreiben Sie das Aufforderungswort

### Nicht zu lange verwenden

DeepSeek ist nicht dasselbe wie ChatGPT.

ChatGPT schreibt eine lange Liste von Aufforderungswörtern und liest sie alle.

Wenn Sie DeepSeek zu lange schreiben, gerät es leicht aus der Bahn.

**Richtige Schreibweise**:
„
Schreiben Sie einen Artikel über Kommunikation am Arbeitsplatz, etwa 1.000 Wörter. Die Zielgruppe sind Berufseinsteiger. Es sollte praktisch und nicht zu theoretisch sein.
„

**Falsche Schreibweise**:
„
Sie sind ein erfahrener Personalexperte mit 10 Jahren Erfahrung in der Personalbeschaffung und können gut mit Beziehungsproblemen am Arbeitsplatz umgehen. Bitte schreiben Sie einen Artikel nach folgenden Vorgaben: erstens... zweitens... dann... schließlich...
„

Es ist zu lang, DeepSeek hat das Ende gelesen und den Anfang vergessen.

### Fragen Sie Schritt für Schritt

Stellen Sie komplexe Fragen auseinander.

Wenn Sie beispielsweise einen Artikel schreiben möchten:

Schritt eins:
„
Helfen Sie mir, einen Überblick über den Artikel „Kommunikationsfähigkeiten am Arbeitsplatz“ (5 Teile) zu erstellen
„

Nachdem DeepSeek den Überblick gegeben hatte:

Schritt zwei:
„
Schreiben Sie basierend auf der ersten Gliederung einen detaillierten Absatz mit 200 Wörtern
„

Schritt drei:
„
Fahren Sie mit dem Schreiben des zweiten Gliederungsteils fort
„

Das funktioniert viel besser, als alle Fragen auf einmal zu stellen.

### Format löschen

DeepSeek kann es manchmal übertreiben und eine Menge Dinge schreiben, die Sie nicht brauchen.

Sagen Sie ihm genau, welches Format Sie wünschen.

„
Bitte stellen Sie es tabellarisch dar. Jedes Vergleichselement sollte Folgendes umfassen: Funktion, DeepSeek-Leistung, ChatGPT-Leistung
„

„
Bitte erläutern Sie in Punkten, nicht mehr als 5 Elemente, jedes Element sollte 50 Wörter nicht überschreiten.
„

Das Spielen wird nicht kostenlos sein.

## Tipp 3: Szenariobasierter Einsatz

### Kopie schreiben

DeepSeek ist sehr gut darin, chinesische Texte zu verfassen.

Vor allem die bodenständige Art.

Zum Beispiel einen Text für Moments schreiben:
„
Helfen Sie mir, einem Freundeskreis zu schreiben, dass meine Kollegen mir heute Nachmittag Milchtee und Kuchen zum Nachmittagstee geschenkt haben. Sei ein bisschen lustig.
„

DeepSeek bietet Ihnen eine interessante Version.

### Code schreiben

Es kann auch beim Schreiben von Code verwendet werden.

Aber der tatsächlich gemessene Effekt ist nicht so gut wie bei Claude.

Claudes Code-Schreiben ist standardisierter und seine Kommentare sind klarer.

DeepSeek hat manchmal kleinere Probleme beim Schreiben von Code, daher wird empfohlen, es zweimal zu überprüfen.

###Schreiben Sie einen Artikel

DeepSeek kann offizielle Account-Artikel, Zhihu-Antworten und Xiaohongshu-Texte verfassen.

Insbesondere bei Inhalten, die „Internet-Sinn“ erfordern, ist das Sprachverständnis von DeepSeek sehr gut.

### übersetzen

Die Übersetzung von DeepSeek ist natürlicher als die von Google Translate.

Es ist keine starre maschinelle Übersetzung, aber sie liest sich wie menschliche Sprache.

„
Bitte helfen Sie mir, diesen Absatz vom Englischen ins Chinesische zu übersetzen. Es sollte natürlich und nicht wörtlich sein.
„

Es funktioniert großartig.

### Brainstorming

Fällt Ihnen kein passendes Thema ein? Sie wissen nicht, wie Sie eine Entscheidung treffen sollen?

Lassen Sie sich von DeepSeek beim Brainstorming unterstützen.

„
Ich bin ein Blogger am Arbeitsplatz. Ich schreibe hauptsächlich über berufliches Wachstum und Kommunikationsfähigkeiten. Bitte geben Sie mir 10 Themen zur Auswahl.
„

Es wird Ihnen eine Menge Ideen geben.

## Tipp 4: Leitfaden zur Vermeidung von Fallstricken

### Verlassen Sie sich nicht zu sehr

DeepSeek redet manchmal ernsthaften Unsinn.

Es fabriziert Daten und Zitate, um professionell auszusehen, ist aber in Wirklichkeit eine Fälschung.

**Lösung**:
- Manuelle Überprüfung wichtiger Informationen
- Umfasst die Überprüfung des Fachwissens aus mehreren Blickwinkeln
- Kopieren Sie nicht direkt in das offizielle Dokument

### Sensible Themen

DeepSeek verfügt über eine Inhaltsüberprüfung und es gibt einige Fragen, die es nicht beantwortet.

Zum Beispiel:
- So umgehen Sie die Zahlung
- Wie man Software knackt
- Sensible historische Ereignisse

Es wird diese Fragen ablehnen oder Ihnen einen offiziellen Vortrag halten.

### Kontextuelle Einschränkungen

Das Kontextfenster von DeepSeek ist nicht so lang wie das von Claude.

Am Ende eines langen Artikels vergessen Sie möglicherweise, was zuvor gesagt wurde.

**Lösung**:
- Schreiben Sie in Abschnitten. Jeder Abschnitt sollte maximal 2.000 Wörter lang sein.
- Wiederholen Sie wichtige Informationen am Anfang jedes Absatzes
- Fassen Sie den vorherigen Inhalt manuell zusammen und füttern Sie ihn bei Bedarf

### Fragen Sie nicht nach Dingen, die zu neu sind.

Die Trainingsdaten von DeepSeek haben ein Ablaufdatum.

Es weiß möglicherweise nicht, was nach 2024 passiert.

Wenn Sie es fragen: „Was für eine große Sache ist in letzter Zeit passiert?“, erhalten Sie eine veraltete Antwort.

## Tipp 5: Erweiterte Nutzung

### Systemaufforderungswort

DeepSeek unterstützt Systemaufforderungswörter.

Klicken Sie oben rechts auf „Einstellungen“, um das Standardverhalten der KI festzulegen.

Zum Beispiel:
„
Sie sind ein professioneller Unternehmensberater. Beantworten Sie Fragen präzise und effektiv und nutzen Sie Daten und Fälle, um Ihre Meinung zu untermauern.
„

Sobald die Einstellung festgelegt ist, werden alle nachfolgenden Konversationen wirksam.

### Konversationsvorlage

Nützliche Aufforderungswörter können als Vorlagen gespeichert werden.

DeepSeek verfügt derzeit nicht über eine offizielle Vorlagenfunktion, Sie können jedoch:

1. Schreiben Sie es in ein Dokument
2. Bei Bedarf kopieren und einfügen
3. Oder machen Sie einen Screenshot und speichern Sie ihn

### API-Aufrufe

Entwickler können über die API darauf zugreifen.

Die API von DeepSeek ist günstiger als OpenAI.

Die Dokumentation ist klar geschrieben und entwicklerfreundlich.

### Lokale Bereitstellung

Open Source bedeutet, dass Sie das Modell herunterladen und lokal ausführen können.

Konfigurationsanforderungen:
- Grafikkarte mit mindestens 16 GB Videospeicher
- Speicher 32 GB oder mehr
- Festplattenspeicher 100 GB+

Auf normalen Computern kann es nicht ausgeführt werden, daher wird die Nutzung von Cloud-Diensten empfohlen.

## FAQ

**F: Erhebt DeepSeek Gebühren? **

A: Die Webversion ist völlig kostenlos. API-Aufrufe sind ebenfalls günstig, viel günstiger als bei OpenAI.

**F: Was ist besser, DeepSeek oder ChatGPT? **

A: Schauen Sie sich die Szene an. DeepSeek ist besser darin, chinesische Inhalte zu schreiben, und Claude ist besser darin, Code zu schreiben. Sie können es für den täglichen Gebrauch ausprobieren.

**F: Sind DeepSeek-Daten sicher? **

A: Die Daten der Webversion werden auf den Server hochgeladen. Die Daten der lokalen Bereitstellungsversion sind vollständig lokal und sicherer.

**F: Wird DeepSeek Menschen ersetzen? **

A: Nein. KI ist ein Werkzeug, kein Ersatz. Wer den Umgang mit KI lernt, wird diejenigen eliminieren, die sie nicht nutzen können.

**F: Muss DeepSeek die Firewall umgehen? **

A: Nein, es besteht eine direkte Verbindung innerhalb Chinas.

**F: Verfügt DeepSeek über eine App? **

A: Derzeit gibt es nur eine Webversion und keine offizielle App.

**F: Kann DeepSeek Bilder generieren? **

A: Nein. DeepSeek ist ein Textmodell und kann keine Bilder generieren.
