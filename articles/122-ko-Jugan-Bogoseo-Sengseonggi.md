## 서문

매주 보고서를 쓸 때마다, 저는 "쓸데없는 일기"를 쓰는 것 같아요.

"이번 주 XX 업무 완료, 다음 주 XX 예정" — 제 자신을 읽어도 민망해요. 더 민망한 건 300자를 채우기 위해 머리를 쥐어짜야 하고, 상사가 제가 아무것도 안 한다고 생각할까봐 무서워요.

나중에 생각해보니까 저만 그런 게 아니었어요. 주변 친구들이 다 주간 보고서를 고민하고 있어요:
- "주간 보고서가 연애편지처럼 길어"
- "월요일 아침부터 주간 보고서가 걱정돼"
- "온 주일 일했는데 200자도 못 쓰겠어"

그래서 AI 주간 보고서 생성기를 쓰기 시작했어요.

5가지 주요 도구를 테스트한 후, 이 리뷰를 쓰기로 했어요. 이건 광고 아니라 그냥 공유입니다.

## 도구별 결과

### ChatGPT

**결과**:
```
이번 주 업무 요약

I. 주요 업무
1. 프로젝트 진행: 킥오프 회의 참석, Q2 목표 확인
2. 기술 지원: XX 모듈 버그 3개 수정
3. 기능 개발: 신기능 출시, 테스트 통과
4. 팀 협력: 동료 디버깅 도움

II. 다음 주 계획
1. 사용자 피드백 확인
2. 다음 단계 개발 진행
3. 코드 최적화

III. 소감
효과적인 소통이 프로젝트 성공의 열쇠입니다.
```
