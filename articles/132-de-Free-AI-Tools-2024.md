## Kostenlose KI-Tools

### 1. DeepSeek: Der aufsteigende Stern

Das Inlandsmodell, das lately Furore macht. Komplett kostenlos.

**Vorteile:**
- Starkes logisches Denken, besonders für Code und Mathe
- Chinesisch-Verständnis fast auf Claude-Niveau
- Web-Version, kein Download nötig

**Nachteile:**
- Kann in Stoßzeiten voll werden, aber normalerweise wartet man ein paar Sekunden
- Etwas schwächer bei kreativem Schreiben

**Am besten für:**
- Code schreiben, debuggen
- Mathe-Probleme, Datenanalyse
- Technische Dokumentation

### 2. Kimi: Der König der langen Kontexte

Von MoonShot AI, stark bei chinesischen Aufgaben.

**Vorteile:**
- 2 Millionen Token Kontext, lässt andere Modelle hinter sich
- Einfach PDFs und Papers reinwerfen und direkt fragen
- Natürliche chinesische Ausdrücke, kein Maschinenübersetzungs-Feeling

**Nachteile:**
- Schwächeres logisches Denken im Vergleich zu DeepSeek
- Kostenlose Version hat Nutzungslimits

**Am besten für:**
- Papers und Berichte lesen
- Lange Dokumente zusammenfassen
- Chinesisches Copywriting

### 3. Doubao: Das ByteDance-Angebot

Die KI von ByteDance, jetzt komplett kostenlos.

**Vorteile:**
- Schnelle Antwort, rarely hängt
- Viele kleine Tools (Übersetzung, Schreiben, Coding)
- Gute App-Erfahrung

**Nachteile:**
- Kreatives Schreiben ist mittelmäßig
- Nicht gut für tiefe Analysen

**Am besten für:**
- Alltägliche Fragen
- Einfaches Copywriting
- Schnelle Informationsbeschaffung

### 4. ChatGPT 3.5: Der Veteran

Auch wenn die neue Version Geld kostet, ist Version 3.5 immer noch kostenlos.

**Vorteile:**
- Reife Ökosystem, weit bekannt
- Reichhaltige Prompt-Vorlagen
- Starke Englisch-Fähigkeiten

**Nachteile:**
- Chinesisch schwächer als Inlandsmodelle
- Kontoregistrierung nötig

**Am besten für:**
- Englisch schreiben
- Übersetzung
- Allgemeine Fragen

## Echte Nutzungserfahrung

Ich rotiere durch diese 4 Tools. Meine Meinung:

**DeepSeek** lately sehr hot, Code-Fähigkeiten auch solid. Kann aber in Stoßzeiten langsam werden—20-22 Uhr meiden empfohlen.

**Kimi** ist unglaublich für Papers lesen. Zehn Papers reinwerfen und zusammen fragen—viel schneller als einzeln lesen. Schade nur, dass die kostenlose Version Limits hat.

**Doubao** ist einfach eine sorgenfreie Wahl. Schnelle Antwort, keine Komplikationen. Genug für Wochenberichte und kleines Copywriting.

**ChatGPT 3.5** mag alt sein, aber für Englisch-Szenarien immer noch der Beste. Besonders bei technischer Dokumentation—gibt Stack Overflow-artige Antworten.
