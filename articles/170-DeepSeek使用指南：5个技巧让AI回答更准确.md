## 技巧2：提示词怎么写

### 别用太长

DeepSeek跟ChatGPT不太一样。

ChatGPT你写一长串提示词，它能都读完。

DeepSeek你写太长，它容易跑偏。

**正确写法**：
```
写一篇关于职场沟通的文章，1000字左右，目标读者是刚入职的新人，要实用，不要太理论。
```

**错误写法**：
```
你是一位资深的人力资源专家，有10年的招聘经验，擅长处理职场关系问题，请按照以下要求写一篇文章：首先...其次...然后...最后...
```

太长了，DeepSeek读到后面忘了前面。

### 分步骤问

复杂问题拆开问。

比如你想写一篇文章：

第一步：
```
帮我列一个"职场沟通技巧"文章的提纲，5个部分
```

DeepSeek给了提纲后：

第二步：
```
根据第一个提纲，写一段200字的详细内容
```

第三步：
```
继续写第二个提纲部分
```

这样比一次性问所有问题效果好得多。

### 明确格式

DeepSeek有时候会过度发挥，写一堆你不需要的东西。

明确告诉它你要什么格式。

```
请用表格形式展示，每个对比项要包含：功能、DeepSeek表现、ChatGPT表现
```

```
请分点说明，最多5条，每条不超过50字
```

它就不会自由发挥了。

## 技巧3：场景化使用

### 写文案

DeepSeek写中文文案很厉害。

特别是接地气的那种。

比如写朋友圈文案：
```
帮我写一条朋友圈，说今天下午茶被同事投喂了奶茶和蛋糕，要搞笑一点
```

DeepSeek会给你一个有趣的版本。

### 写代码

写代码也能用。

但实测效果不如Claude。

Claude写代码更规范，注释更清晰。

DeepSeek写代码有时候会有小问题，建议二次检查。

### 写文章

公众号文章、知乎回答、小红书文案，DeepSeek都能写。

特别是需要"网感"的内容，DeepSeek的语感很好。

### 翻译

DeepSeek的翻译比Google翻译自然。

不是那种生硬的机器翻译，而是读起来像人话。

```
帮我把这段英文翻译成中文，要自然，不要直译
```

效果很好。

### 头脑风暴

想不出选题？不知道怎么做决策？

让DeepSeek帮你头脑风暴。

```
我是一个职场博主，主要写职场成长和沟通技巧，请给我10个选题方向
```

它会给你一堆ideas。

## 技巧4：避坑指南

### 别太依赖

DeepSeek有时候会一本正经地胡说八道。

它编造数据、编造引用，看起来很专业，其实是假的。

**解决方法**：
- 重要信息手动核实
- 涉及专业知识多角度验证
- 别直接copy到正式文档

### 敏感问题

DeepSeek有内容审查，有些问题它不回答。

比如：
- 怎么绕过付费
- 怎么破解软件
- 敏感历史事件

这些问题它会拒绝，或者给你一段官话。

### 上下文限制

DeepSeek的上下文窗口不如Claude长。

长文章写到后面，它可能忘了前面说了什么。

**解决方法**：
- 分段写，每段控制在2000字以内
- 重要信息在每段开头重复一下
- 必要时手动总结前面内容喂给它

### 别问太新的事

DeepSeek的训练数据有截止日期。

2024年之后的事它可能不知道。

问它"最近发生了什么大事"，它会给一个过时的回答。

## 技巧5：进阶用法

### 系统提示词

DeepSeek支持系统提示词。

点击右上角设置，可以设置AI的默认行为。

比如：
```
你是一位专业的商业顾问，回答问题要简洁有力，多用数据和案例支持观点。
```

设置一次，后面所有对话都生效。

### 对话模板

好用的提示词可以存成模板。

DeepSeek目前没有官方模板功能，但你可以：

1. 写在一个文档里
2. 要用的时候复制粘贴
3. 或者截图保存

### API调用

开发者可以用API接入。

DeepSeek的API比OpenAI便宜。

文档写得很清楚，开发者友好。

### 本地部署

开源意味着你可以下载模型本地跑。

配置要求：
- 显卡至少16GB显存
- 内存32GB以上
- 硬盘空间100GB+

普通电脑跑不动，建议用云服务。
