# AI比一比 (AICompare) - 多AI站点对比神器

> 一个标签页内同时对比多个AI的生成结果，让AI使用效率翻倍！

## 🚀 为什么需要 AI比一比？

在AI工具爆发的时代，你是否也有以下困扰？

- **切换累** - 在ChatGPT、Claude、Perplexity之间反复切换标签页
- **重复问** - 同一个问题要在多个AI上问一遍才能对比
- **效率低** - 等待一个AI回答的同时，错过了其他AI的更快答案
- **选择难** - 不知道哪个AI更适合当前任务

**AI比一比** 解决的就是这些问题！

## ✨ 核心功能

### 1️⃣ 多站对比
一个标签页内嵌入多个AI，输入一次关键词，**同时展示**多个AI的查询结果。横向对比，一目了然。

### 2️⃣ 一键打开
支持一键打开你常用的AI站点：
- ChatGPT
- Claude
- Perplexity
- Kimi
- 豆包
- POE
- Gemini
- ...持续增加中

### 3️⃣ 快捷键触发
按 `⌘+M` (Mac) 或 `Ctrl+M` (Windows) 快速唤醒，输入关键词后选择要打开的站点。

### 4️⃣ 划词查询
浏览网页时，鼠标选中文字，旁边会出现操作图标，点击即可快速让多个AI帮你处理选中的内容。

### 5️⃣ 标签页模式
高频用户可以开启「标签页模式」，每次新建标签页自动打开多AI对比视图。
